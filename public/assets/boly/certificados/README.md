# boly / certificados

## Qué va en esta carpeta

Boly™ en certificados, logros e insignias. Esta carpeta está reservada únicamente para activos reales aprobados de PanaderIA™.

## Cómo nombrar archivos

Usa nombres en minúsculas, sin espacios, con guiones medios y número de versión:

`boly-certificado-v001.png`

Estructura recomendada:

`tema-descripcion-contexto-v###.extension`

## Formatos recomendados

PNG transparente, SVG. Mantén una copia fuente editable cuando exista y una versión optimizada para web/producción.

## Cómo evitar duplicados

- Revisa esta carpeta antes de agregar un activo nuevo.
- Si el archivo representa el mismo contenido con cambios menores, incrementa la versión en lugar de crear un nombre distinto.
- No subas capturas temporales, pruebas ni archivos sin aprobación.
- Conserva el nombre semántico aunque cambie la extensión.

## Cómo saber cuál es la versión oficial

- La versión oficial debe incluir `oficial` cuando sea el activo maestro de identidad.
- La versión vigente será la de número `v###` más alto que esté aprobada por dirección de marca.
- Cuando exista un documento de control de activos, registra ahí la ruta, versión, fecha y responsable.
- No borres versiones anteriores sin autorización; márcalas como históricas en el control documental cuando sea necesario.
