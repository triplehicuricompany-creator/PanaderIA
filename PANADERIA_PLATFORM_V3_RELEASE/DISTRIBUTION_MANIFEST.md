# Manifiesto del paquete de distribución

## Incluye

- Plataforma estática lista para hosting.
- Curso Maestro de Bolillo completo en JSON.
- Biblioteca técnica y buscador local.
- Motor local Boly™.
- Recursos descargables por módulo.
- Videos de referencia por módulo.
- Certificados y productos comerciales preparados.
- Manuales operativos para administradores, instructores y estudiantes.
- Checklist previo y posterior al lanzamiento.
- Plan de respaldo y recuperación.

## No incluye

- Backend real de pagos.
- Base de datos de usuarios.
- Autenticación de producción.
- Videos finales grabados del Chef Juanote.
- PDFs diseñados finales.

Estos elementos están preparados para integración mediante las rutas y mapas incluidos.
