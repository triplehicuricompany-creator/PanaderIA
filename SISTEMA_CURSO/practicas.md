# Prácticas y Ejercicios

## Propósito

Definir el marco administrativo para prácticas y ejercicios futuros sin crear instrucciones finales, hojas descargables ni materiales de alumno.

## Tipos de práctica permitidos como categoría

- Práctica técnica.
- Práctica de observación.
- Práctica de diagnóstico.
- Práctica de repetición controlada.
- Práctica integradora.
- Ejercicio de autoevaluación.

## Ficha estructural de práctica

| Campo | Uso |
|---|---|
| `id_practica` | Identificador único. |
| `id_modulo` | Módulo relacionado. |
| `id_leccion` | Lección relacionada, si aplica. |
| `tipo` | Categoría de práctica. |
| `evidencia_esperada` | Campo reservado para fase futura. |
| `criterios_revision` | Campo reservado para fase futura. |
| `estado` | Estado documental. |

## Restricciones

- No redactar pasos de práctica.
- No crear hojas de trabajo.
- No crear descargables.
- No generar rúbricas finales.
