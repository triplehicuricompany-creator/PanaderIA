# Índice Maestro del Curso

## Propósito

Definir el índice maestro como estructura de control del Curso Maestro Premium PanaderIA™ sin generar contenido educativo.

## Estructura del índice

El índice futuro deberá organizarse por:

1. Nivel del curso.
2. Módulo.
3. Lección.
4. Práctica asociada.
5. Evaluación asociada.
6. Certificación o hito aplicable.
7. Estado documental.

## Campos de control

| Campo | Descripción |
|---|---|
| `orden` | Secuencia administrativa. |
| `id_elemento` | Identificador del módulo, lección o evaluación. |
| `tipo` | Módulo, lección, práctica, evaluación o certificación. |
| `estado` | Estado documental vigente. |
| `fuente` | Documento canónico autorizado para desarrollo futuro. |
| `observaciones` | Notas de integración futura. |

## Restricción

Este índice no debe convertirse en temario desarrollado hasta que exista autorización explícita en una fase posterior.
