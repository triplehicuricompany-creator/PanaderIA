# Biblioteca de conocimiento de Boly™ — Curso Maestro de Bolillo

Esta carpeta queda preparada para la integración futura de Boly™ como asistente educativo dentro de cada módulo del Curso Maestro de Bolillo.

## Autor

Chef Juanote

## Frases guía

- "No hay mejor maestro que el error mismo"
- "Nunca has fracasado si ya has comenzado"

## Estructura por módulo

Boly™ deberá tener contexto específico para:

1. Introducción al Bolillo
2. Herramientas necesarias
3. Ingredientes
4. Amasado
5. Fermentación
6. División y boleado
7. Formado del bolillo
8. Corte correcto
9. Horneado y vapor
10. Enfriado
11. Bolillos especiales
12. Problemas comunes
13. Evaluación
14. Certificación

Cada módulo podrá enlazar:

- Fotografías técnicas
- Videos QR
- PDF descargable
- Material complementario
- Preguntas frecuentes
- Ayuda contextual de Boly™

En esta fase no se conecta IA real ni base de datos vectorial.
