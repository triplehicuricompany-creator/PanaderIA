# 02_GUIONES

## Función

Guiones literarios, escaletas, storyboards textuales y versiones aprobadas por capítulo.

## Qué archivos deben almacenarse

- Archivos finales o de trabajo relacionados únicamente con esta categoría multimedia.
- Documentos de apoyo, notas de producción y evidencias de aprobación cuando apliquen.
- Versiones con fecha, responsable, estado y vínculo al capítulo o recurso correspondiente.

## Buenas prácticas

- No modificar el Libro Maestro, el Motor Maestro, la Biblioteca Visual IA ni los Recursos del Alumno desde esta carpeta.
- Mantener a Boly como asistente oficial para orientación al alumno y al Chef Juanote como autoridad técnica.
- Separar borradores, aprobados y finales mediante nombres de archivo claros.
- Registrar recursos relevantes en `../manifest.json` antes de publicarlos.
- Validar derechos de uso, licencias, calidad técnica y coherencia educativa antes de exportar.
- Usar lenguaje claro, cálido, profesional, mexicano y apto para principiantes.

## Convención de nombres

```text
panaderia-02_guiones-cap[00]-[tema]-[formato]-[estado]-v[00].[ext]
```

Ejemplo:

```text
panaderia-02_guiones-cap03-fermentacion-video-final-v01.mp4
```

Estados sugeridos: `borrador`, `revision`, `aprobado`, `final`, `publicado`.

## Flujo de trabajo recomendado

1. Identificar el capítulo, objetivo didáctico y alumno destinatario.
2. Crear o recibir el archivo en estado `borrador`.
3. Revisar precisión técnica con Chef Juanote cuando el contenido enseñe procesos de panadería.
4. Revisar claridad, tono y acompañamiento con Boly cuando el contenido guíe al alumno.
5. Guardar la versión aprobada con nomenclatura oficial.
6. Registrar ubicación, uso, licencia y estado en el manifiesto multimedia.
7. Mover o copiar la versión lista a la carpeta de publicación correspondiente.
