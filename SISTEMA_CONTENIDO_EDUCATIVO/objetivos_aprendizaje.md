# Objetivos de Aprendizaje

## Propósito

Definir el marco oficial para formular, clasificar y validar objetivos de aprendizaje dentro del ecosistema educativo PanaderIA™.

## Principios

- Todo objetivo debe ser observable, medible y vinculado a una unidad canónica.
- Los objetivos deben orientar ejercicios, evaluaciones y recursos de apoyo.
- Este archivo no redacta objetivos finales de capítulos; solo establece la gobernanza para crearlos después.

## Estructura recomendada

| Campo | Descripción |
|---|---|
| `id_objetivo` | Identificador estable del objetivo. |
| `unidad_referencia` | Capítulo, módulo o lección asociada. |
| `verbo_accion` | Acción observable esperada. |
| `competencia` | Competencia técnica, conceptual o profesional. |
| `evidencia` | Resultado verificable esperado. |
| `nivel_taxonomico` | Reconocer, aplicar, analizar, evaluar o crear. |
| `criterio_validacion` | Condición mínima para aceptar el logro. |

## Categorías de competencias

- Técnicas panaderas.
- Conceptuales y científicas.
- Operativas de producción.
- Diagnóstico y solución de problemas.
- Seguridad, higiene y calidad.
- Gestión profesional y escalamiento.

## Control de calidad

Todo objetivo deberá revisarse para confirmar que no genera contenido final del libro, no altera capítulos y no contradice el sistema de contenido existente.
