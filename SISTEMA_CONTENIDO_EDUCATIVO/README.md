# Sistema Maestro de Contenido Educativo PanaderIA™

El **Sistema Maestro de Contenido Educativo PanaderIA™** es la capa oficial de gobernanza académica para organizar, controlar y escalar el contenido educativo del ecosistema PanaderIA™.

Este sistema complementa al Sistema Maestro de Contenido existente: no reemplaza su función editorial, no duplica sus mapas de contenido y no genera contenido final del libro. Su responsabilidad es establecer criterios educativos, estructuras de seguimiento, estados documentales y reglas de coordinación para capítulos, lecciones, objetivos, ejercicios, evaluaciones, recursos de apoyo y participaciones pedagógicas de Boly y del Chef Juanote.

## Alcance

- Definir la arquitectura de gobernanza educativa.
- Organizar capítulos únicamente como unidades académicas de referencia.
- Organizar lecciones como unidades didácticas controladas.
- Declarar el marco de objetivos de aprendizaje.
- Establecer reglas para ejercicios y prácticas sin desarrollarlos.
- Establecer reglas para evaluaciones sin redactar exámenes finales.
- Definir recursos de apoyo como inventario estructural futuro.
- Gobernar la participación educativa de Boly.
- Gobernar la participación educativa del Chef Juanote.
- Mantener un roadmap de crecimiento del sistema educativo.

## Restricciones principales

- No modificar capítulos existentes.
- No duplicar el Sistema Maestro de Contenido existente.
- No generar contenido final del libro.
- No escribir capítulos nuevos.
- No crear PDFs.
- No crear videos.
- No crear descargables finales.

## Archivos del sistema

| Archivo | Propósito |
|---|---|
| `manifest.json` | Declaración oficial del sistema, alcance, restricciones y estados. |
| `capitulos.md` | Gobernanza de capítulos como unidades educativas de referencia. |
| `lecciones.md` | Organización estructural de lecciones sin redactarlas. |
| `objetivos_aprendizaje.md` | Marco para objetivos de aprendizaje, competencias y evidencias. |
| `ejercicios.md` | Reglas para prácticas y ejercicios futuros sin producir contenido final. |
| `evaluaciones.md` | Sistema de control evaluativo sin crear exámenes finales. |
| `recursos_apoyo.md` | Inventario estructural de recursos de apoyo futuros. |
| `contenido_boly.md` | Lineamientos para participación educativa de Boly. |
| `contenido_chef_juanote.md` | Lineamientos para participación educativa del Chef Juanote. |
| `roadmap.md` | Plan de crecimiento de la gobernanza educativa. |

## Flujo maestro de gobernanza

1. Identificar la unidad educativa sin alterar contenido existente.
2. Registrar su relación con capítulo, lección, objetivo, ejercicio, evaluación y recurso de apoyo.
3. Asignar estado documental y responsable futuro.
4. Validar que no se esté generando contenido final ni duplicando sistemas existentes.
5. Coordinar futuras integraciones con curso, libro, Boly, Chef Juanote, multimedia y publicación.
6. Actualizar el roadmap y dejar trazabilidad de decisiones académicas.

## Estado de esta misión

Esta carpeta crea únicamente la infraestructura documental oficial del Sistema Maestro de Contenido Educativo PanaderIA™. Todo contenido académico específico queda marcado como pendiente de integración futura.
