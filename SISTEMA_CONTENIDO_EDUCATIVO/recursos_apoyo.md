# Recursos de Apoyo

## Propósito

Definir la gobernanza de recursos de apoyo educativo futuros sin crear archivos descargables, PDFs, videos, imágenes finales ni materiales publicables.

## Categorías estructurales

- Glosarios de apoyo.
- Checklists futuros.
- Fichas técnicas futuras.
- Mapas visuales futuros.
- Recursos de diagnóstico.
- Materiales de refuerzo.
- Guías de acompañamiento.
- Referencias internas del ecosistema PanaderIA™.

## Ficha de control

| Campo | Descripción |
|---|---|
| `id_recurso` | Identificador del recurso futuro. |
| `categoria` | Tipo de recurso de apoyo. |
| `unidad_relacionada` | Capítulo, lección, objetivo o evaluación vinculada. |
| `uso_previsto` | Función educativa esperada. |
| `formato_previsto` | Formato futuro, no generado en esta fase. |
| `estado` | Estado documental. |
| `restricciones` | Alertas de no producción o no publicación. |

## Reglas de control

- Los recursos se registran como infraestructura, no como entregables finales.
- Todo recurso debe tener una unidad educativa vinculada.
- Los formatos multimedia o descargables deben producirse en sistemas especializados.
- Ningún recurso de esta carpeta debe exportarse como PDF o video.
