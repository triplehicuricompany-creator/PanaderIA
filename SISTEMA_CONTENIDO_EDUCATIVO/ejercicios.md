# Ejercicios y Prácticas

## Propósito

Establecer el marco de gobernanza para ejercicios y prácticas educativas futuras sin crear actividades finales, hojas descargables ni instrucciones operativas completas.

## Tipos de ejercicios permitidos como estructura

- Diagnóstico conceptual.
- Observación técnica.
- Práctica guiada futura.
- Autoevaluación.
- Corrección de errores.
- Simulación de producción.
- Registro de evidencias.

## Ficha estructural

| Campo | Descripción |
|---|---|
| `id_ejercicio` | Identificador de práctica futura. |
| `objetivo_relacionado` | Objetivo de aprendizaje vinculado. |
| `tipo` | Categoría del ejercicio. |
| `evidencia_esperada` | Evidencia general sin detallar contenido final. |
| `criterio_revision` | Criterio académico futuro. |
| `riesgos` | Riesgos de seguridad, calidad o interpretación. |
| `estado` | Estado documental. |

## Restricciones

- No escribir instrucciones finales paso a paso.
- No generar fichas descargables.
- No crear plantillas listas para publicar.
- No sustituir prácticas oficiales del curso existente.

## Integración futura

Los ejercicios solo podrán pasar a producción cuando cuenten con capítulo de referencia, objetivo aprobado, criterio de evaluación y revisión de seguridad.
