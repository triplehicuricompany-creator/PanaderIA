# Identidad visual

## Objetivo

Definir la infraestructura documental para gobernar la identidad visual de marketing sin crear artes finales.

## Componentes a documentar en fases futuras

- Logotipo y versiones permitidas.
- Paleta cromática.
- Tipografías.
- Iconografía.
- Fotografía.
- Ilustraciones.
- Uso de personajes.
- Plantillas para redes, email, presentaciones y campañas.

## Criterios visuales

La identidad debe transmitir calidez panadera, orden educativo, profesionalismo, claridad y confianza.

## Reglas

- No deformar marca, personajes o recursos visuales.
- No usar estilos visuales que contradigan el tono educativo.
- No publicar artes sin revisión de marca.
- No crear piezas finales desde este documento.
