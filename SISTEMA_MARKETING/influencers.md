# Influencers y colaboraciones

## Objetivo

Establecer criterios para colaboraciones futuras con creadores, chefs, panaderos, escuelas, marcas o comunidades.

## Criterios de selección

- Afinidad con el oficio panadero.
- Reputación y conducta pública.
- Calidad educativa.
- Audiencia relevante.
- Compatibilidad con valores PanaderIA™.
- Transparencia comercial.

## Tipos de colaboración

- Reseñas autorizadas.
- Contenido educativo conjunto.
- Lives o talleres.
- Afiliación.
- Embajadores.
- Alianzas institucionales.

## Restricciones

No se pactan acuerdos reales, pagos, mensajes, guiones ni publicaciones desde este documento.
