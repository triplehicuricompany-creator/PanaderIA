# Comunidad

## Objetivo

Definir lineamientos para administrar una comunidad futura de PanaderIA™ con seguridad, claridad y valor educativo.

## Principios

- Respeto entre miembros.
- Aprendizaje práctico.
- Moderación humana.
- Cuidado de principiantes.
- Protección de datos y testimonios.
- Separación entre ayuda educativa y soporte comercial.

## Elementos documentales

- Propósito de la comunidad.
- Reglas de participación.
- Roles de moderación.
- Protocolo de conflictos.
- Criterios para destacar miembros.
- Uso de testimonios.
- Relación con lanzamientos y contenidos.

## Restricción

Este documento no crea ni abre una comunidad real.
