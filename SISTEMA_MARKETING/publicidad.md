# Publicidad pagada

## Objetivo

Establecer lineamientos para futuras campañas pagadas sin redactar anuncios ni activar inversión.

## Elementos documentales requeridos

- Objetivo publicitario.
- Audiencia.
- Canal.
- Presupuesto estimado.
- Etapa del embudo.
- Oferta o activo autorizado.
- Mensajes permitidos.
- Claims restringidos.
- Métricas.
- Riesgos.

## Canales posibles

- Meta Ads.
- Google Ads.
- YouTube Ads.
- TikTok Ads.
- LinkedIn Ads.
- Plataformas educativas o de afiliación aprobadas.

## Restricciones

No se escriben anuncios reales, no se crean creatividades finales, no se configuran campañas y no se invierte presupuesto desde esta fase.
