# Publicidad

## Objetivo

Usar campañas pagadas para amplificar el alcance, captar prospectos y generar conversiones de forma controlada y medible.

## Canales potenciales

- Meta Ads para Instagram y Facebook.
- TikTok Ads para descubrimiento y alcance.
- Google Search Ads para intención activa.
- YouTube Ads para autoridad y remarketing.

## Tipos de campaña

1. Reconocimiento de marca.
2. Captación de leads.
3. Tráfico a página de venta.
4. Remarketing.
5. Conversión directa.

## Estructura mínima de prueba

- Hipótesis de campaña.
- Audiencia definida.
- Creativo y mensaje principal.
- Presupuesto de prueba.
- Métrica primaria.
- Criterio de pausa, ajuste o escalamiento.

## Reglas de comunicación pagada

- No prometer resultados garantizados.
- No usar contenido definitivo del libro como anuncio principal sin aprobación futura.
- Validar mensajes con baja inversión antes de escalar.
- Separar campañas de captación, venta y remarketing.
