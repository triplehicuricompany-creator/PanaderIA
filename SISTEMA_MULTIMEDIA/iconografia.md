# Iconografía PanaderIA™

## Propósito

Establecer el marco documental para un sistema de íconos, pictogramas, señales y marcadores visuales coherente con la identidad PanaderIA™.

## Familias previstas

- Íconos de procesos de panadería.
- Íconos de ingredientes.
- Íconos de herramientas y equipo.
- Señales de advertencia, consejo, error y solución.
- Marcadores educativos.
- Íconos de navegación para plataforma.
- Símbolos comerciales o institucionales.

## Principios de consistencia

- Formas simples y legibles.
- Estilo visual unificado.
- Uso controlado de color.
- Significado claro sin ambigüedad.
- Compatibilidad con formatos digitales e impresos.
- Accesibilidad visual en tamaños pequeños.

## Registro futuro

Cada ícono deberá documentar nombre, categoría, significado, uso permitido, variaciones, versión, fuente editable, formato exportable, licencia y estado.

## Exclusiones de esta fase

No se diseñan íconos, no se exportan archivos gráficos y no se modifica ningún sistema visual existente.
