# Organización de videos

## Tipos oficiales de video

- **Video educativo:** explica una técnica, concepto o proceso del curso.
- **Video demostrativo:** muestra una acción práctica paso a paso.
- **Video de diagnóstico:** enseña errores, causas y correcciones.
- **Video comercial:** promueve el Libro Maestro Premium PanaderIA™ o el curso.
- **Video de bienvenida:** presenta módulos, rutas de aprendizaje o uso de plataforma.
- **Video corto:** cápsula vertical para redes, recordatorio o refuerzo.

## Organización recomendada

```text
MULTIMEDIA_ACTIVOS/videos/
├── educativos/
├── demostrativos/
├── diagnostico/
├── comerciales/
├── bienvenida/
├── cortos_verticales/
└── masters/
```

## Resoluciones recomendadas

| Uso | Resolución | Relación | Formato recomendado |
| --- | --- | --- | --- |
| Curso principal | 1920 x 1080 | 16:9 | MP4 H.264 o H.265 |
| Master premium | 3840 x 2160 | 16:9 | ProRes, MOV o MP4 alta tasa |
| Redes verticales | 1080 x 1920 | 9:16 | MP4 H.264 |
| Cuadrado social | 1080 x 1080 | 1:1 | MP4 H.264 |
| Miniatura animada | 1280 x 720 | 16:9 | GIF, MP4 corto o WebM |

## Formatos permitidos

- **Edición/master:** MOV, ProRes, MP4 de alta tasa, proyecto editable del editor usado.
- **Entrega:** MP4 H.264, MP4 H.265.
- **Web interactivo:** WebM cuando sea requerido por la plataforma.
- **Subtítulos:** SRT, VTT.

## Reglas de producción

1. Todo video debe tener objetivo didáctico o comercial explícito.
2. El audio debe ser claro y sin ruido que impida entender instrucciones.
3. La toma debe mostrar manos, masa, herramientas o resultado cuando la técnica lo requiera.
4. Incluir subtítulos o guion transcrito cuando el recurso sea educativo.
5. Evitar música que compita con la voz.
6. Validar que no aparezcan datos personales, marcas no autorizadas o elementos ajenos al sistema.
7. Conservar master editable y exportación final.

## Registro mínimo futuro

- Nombre canónico.
- Duración.
- Capítulo o uso destino.
- Guion asociado.
- Resolución.
- Formato.
- Estado de revisión.
- Enlace o ruta del archivo final.
