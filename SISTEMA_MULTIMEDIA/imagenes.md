# Organización de Imágenes PanaderIA™

## Propósito

Establecer criterios oficiales para administrar imágenes futuras, incluyendo fotografías, ilustraciones, infografías, portadas, banners, mockups y recursos visuales de apoyo.

## Categorías previstas

- Fotografías reales de panadería, procesos, ingredientes y resultados.
- Ilustraciones educativas de apoyo.
- Infografías técnicas o comerciales.
- Portadas, banners y piezas de presentación.
- Recursos para plataforma web y materiales descargables.
- Imágenes provisionales para planeación interna.

## Criterios de organización

Cada imagen futura deberá clasificarse por:

- Uso principal.
- Relación con producto o sistema.
- Formato y resolución.
- Orientación visual.
- Estado de aprobación.
- Derechos y restricciones.
- Versión o edición.

## Reglas visuales

- Mantener coherencia con la identidad PanaderIA™.
- Evitar imágenes engañosas o incompatibles con procesos reales.
- Diferenciar claramente material provisional de material aprobado.
- Registrar fuente, autoría y permiso antes de cualquier publicación.

## Exclusiones de esta fase

No se generan imágenes, no se editan recursos visuales existentes y no se crean archivos multimedia.
