# Organización de imágenes y fotografías

## Categorías oficiales

- **Ilustraciones:** recursos gráficos creados para explicar conceptos, técnicas o procesos.
- **Fotografías de producto:** bolillos, piezas terminadas, miga, corteza, charolas y empaques.
- **Fotografías de proceso:** amasado, reposo, formado, corte, fermentación, horneado y enfriado.
- **Fotografías de ingredientes:** harina, levadura, sal, agua, masas madre, aditivos permitidos y materias primas.
- **Fotografías de equipo:** horno, báscula, amasadora, charolas, cuchillas, termómetros y utensilios.
- **Diagramas e infografías:** líneas de tiempo, mapas de fermentación, matrices de errores y flujos de trabajo.
- **Imágenes comerciales:** banners, portadas, miniaturas y materiales de promoción.

## Organización recomendada

```text
├── ingredientes/
├── equipo/
└── contexto/
```

## Reglas para fotografías

1. Usar luz limpia, uniforme y sin dominantes extremas.
2. Evitar fondos sucios, distractores visuales o marcas no autorizadas.
3. Mostrar pan, masa y utensilios con intención didáctica clara.
4. Mantener consistencia entre sesiones del mismo capítulo.
5. Conservar archivo original y versión editada.
6. No sobreeditar color, textura o volumen del pan hasta volverlo irreal.
7. Incluir referencia de escala cuando ayude a explicar tamaño, greñado o volumen.
8. Documentar si la fotografía es de producto final, proceso real o puesta editorial.

## Resoluciones recomendadas

| Uso | Resolución mínima | Resolución ideal | Notas |
| --- | --- | --- | --- |
| Libro impreso | 300 ppp al tamaño final | 300 ppp o superior | Preferir TIFF o PNG maestro. |
| Pantalla web | 1600 px lado largo | 2400 px lado largo | Optimizar peso para carga rápida. |
| Miniatura | 800 x 450 px | 1280 x 720 px | Mantener legibilidad en móvil. |
| Redes verticales | 1080 x 1920 px | 1440 x 2560 px | Adaptar encuadre, no recortar información clave. |
| Portadas | 2400 px lado largo | 4000 px lado largo | Conservar espacio para títulos. |

## Formatos permitidos

- **Maestros editables:** PSD, AI, SVG, TIFF.
- **Finales de alta calidad:** TIFF, PNG, JPG de alta calidad.
- **Web optimizado:** WebP, JPG, PNG.
- **Diagramas vectoriales:** SVG, PDF, AI.

## Metadatos mínimos

Cada imagen futura debe registrar:

- Nombre canónico.
- Categoría.
- Capítulo o sección destino.
- Autor, fuente o responsable.
- Estado: borrador, revisión, aprobado o reemplazado.
- Derechos de uso.
- Fecha de creación o incorporación.
