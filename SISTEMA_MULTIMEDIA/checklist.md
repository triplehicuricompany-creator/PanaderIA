# Checklist de control de calidad multimedia

## Revisión general

- [ ] El recurso tiene propósito claro.
- [ ] El archivo usa la nomenclatura oficial.
- [ ] La categoría multimedia es correcta.
- [ ] El recurso está asociado a capítulo, módulo, área o uso comercial.
- [ ] El formato es permitido.
- [ ] La resolución o calidad técnica cumple el estándar.
- [ ] Existen derechos de uso claros.
- [ ] Se conserva master editable cuando aplica.
- [ ] El recurso no modifica capítulos existentes por sí mismo.

## Imágenes y fotografías

- [ ] La imagen está enfocada y bien iluminada.
- [ ] El color representa fielmente pan, masa o ingredientes.
- [ ] No hay fondos, objetos o marcas no autorizadas.
- [ ] La resolución sirve para impresión o web según destino.
- [ ] El encuadre comunica la información necesaria.

## Videos

- [ ] El audio es claro.
- [ ] La imagen está estable y bien expuesta.
- [ ] La duración es adecuada para el objetivo.
- [ ] Existen subtítulos o transcripción cuando corresponde.
- [ ] La exportación fue revisada completa antes de aprobarse.

## Códigos QR

- [ ] El QR abre el destino correcto.
- [ ] El QR fue probado en al menos un dispositivo móvil.
- [ ] El tamaño impreso o digital es legible.
- [ ] La URL destino está registrada.
- [ ] La versión y fecha de prueba están documentadas.

## Audios

- [ ] No hay saturación.
- [ ] El ruido de fondo es aceptable.
- [ ] El volumen es consistente.
- [ ] La voz se entiende con claridad.
- [ ] El formato coincide con el destino.

## Animaciones

- [ ] La animación explica o refuerza un objetivo real.
- [ ] El movimiento no distrae.
- [ ] El peso final es razonable.
- [ ] Se revisó en el dispositivo o plataforma destino.
- [ ] Existe master editable cuando aplica.

## Resultado

- [ ] Aprobado para integración.
- [ ] Requiere corrección.
- [ ] Rechazado.
- [ ] Reemplazado por nueva versión.
