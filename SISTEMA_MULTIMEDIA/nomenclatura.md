# Convención oficial de nombres

## Fórmula maestra

```text
panaderia_[tipo]_[capitulo-o-area]_[tema]_[uso]_[version].[extension]
```

## Campos

- **panaderia:** prefijo obligatorio del ecosistema.
- **tipo:** imagen, foto, video, qr, audio, animacion, infografia, diagrama.
- **capitulo-o-area:** cap01, cap02, bolillo, fermentacion, comercial, plataforma, etc.
- **tema:** palabra breve y descriptiva sin acentos.
- **uso:** libro, web, curso, redes, portada, descarga, evaluacion, interno.
- **version:** v01, v02, final, aprobado, reemplazado.
- **extension:** formato real del archivo.

## Reglas

1. Usar minúsculas.
2. No usar espacios.
3. No usar acentos ni caracteres especiales.
4. Separar palabras con guion bajo.
5. Mantener nombres descriptivos, no genéricos.
6. No usar nombres como `final_final`, `nuevo`, `copia` o `sin_titulo`.
7. Incrementar versión cuando cambie contenido, destino o composición.
8. Usar `aprobado` solo después de control de calidad.

## Ejemplos válidos

```text
panaderia_foto_cap03_amasado_proceso_v01.jpg
panaderia_video_cap07_greñado_curso_final.mp4
panaderia_qr_cap05_evaluacion_web_v01.svg
panaderia_audio_bolillo_intro_curso_v02.wav
panaderia_animacion_fermentacion_expansion_web_final.webm
panaderia_infografia_cap02_harinas_libro_aprobado.pdf
```

## Ejemplos no válidos

```text
foto bonita.jpg
VideoFinalFINAL.mp4
qr nuevo.png
capítulo 3 fermentación!!.jpg
pan bolillo copia 2.png
```
