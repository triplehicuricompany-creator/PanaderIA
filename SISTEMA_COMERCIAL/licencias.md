# Licencias de uso

## Propósito

Establecer el marco para futuras licencias de acceso, uso, reproducción limitada, formación, distribución o integración comercial.

## Tipos de licencia a definir

- Licencia personal.
- Licencia para estudiantes.
- Licencia para instructores.
- Licencia para escuelas o academias.
- Licencia institucional.
- Licencia empresarial.
- Licencia de distribución autorizada.

## Reglas maestras

- Ninguna licencia queda activa por este documento.
- Toda licencia futura debe especificar alcance, duración, territorio, permisos, restricciones y soporte.
- Las licencias deben validarse legalmente antes de publicarse o venderse.
