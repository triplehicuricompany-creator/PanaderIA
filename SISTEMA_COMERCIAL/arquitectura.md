# Arquitectura comercial

## Objetivo

Definir la estructura general que organizará la estrategia comercial de PanaderIA™ a nivel documental, conceptual y operativo.

## Componentes principales

1. **Productos:** ofertas formativas, recursos, experiencias y servicios definidos de forma conceptual.
2. **Clientes:** segmentos, perfiles y necesidades que orientan futuras decisiones comerciales.
3. **Ventas:** principios y canales para convertir interés en adquisición de manera ética.
4. **Embudos:** recorridos comerciales desde descubrimiento hasta fidelización.
5. **Ofertas:** reglas para construir promociones futuras sin redactar mensajes finales.
6. **Suscripciones:** posibles modelos recurrentes de acceso, acompañamiento o recursos.
7. **Afiliados:** criterios para alianzas comerciales controladas.
8. **Roadmap:** secuencia de maduración del sistema comercial.

## Flujo documental

Toda iniciativa comercial futura deberá pasar por una secuencia mínima:

1. Identificación del producto o activo comercial.
2. Relación con un tipo de cliente definido.
3. Selección del canal o embudo aplicable.
4. Definición conceptual de oferta, precio o modalidad.
5. Revisión de coherencia con el ecosistema PanaderIA™.
6. Aprobación antes de producción de contenido definitivo.

## Restricciones

Este documento no crea campañas, no define precios finales, no redacta copys comerciales y no modifica contenidos pedagógicos existentes.
