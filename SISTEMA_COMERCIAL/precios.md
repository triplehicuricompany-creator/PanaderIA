# Precios y versiones

## Propósito

Definir la estructura base para administrar futuras versiones del producto, paquetes comerciales y criterios de precio.

## Versiones a evaluar

- Versión personal.
- Versión estudiante.
- Versión profesional.
- Versión institucional.
- Versión para instructores.
- Versión empresarial o de capacitación interna.
- Paquetes con recursos digitales o certificación.

## Reglas de precio

- No declarar precios finales en esta fase.
- Registrar moneda, país, impuestos, vigencia y condiciones cuando se aprueben precios futuros.
- Documentar descuentos, becas o promociones como reglas controladas y no como excepciones informales.
