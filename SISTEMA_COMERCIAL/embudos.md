# Embudos comerciales

## Objetivo

Mapear recorridos comerciales posibles sin crear campañas, anuncios, correos, páginas ni piezas definitivas.

## Etapas base

1. **Descubrimiento:** el cliente conoce PanaderIA™ o identifica un problema panadero.
2. **Educación:** recibe contexto, diagnóstico o contenido introductorio.
3. **Evaluación:** compara alternativas y valida pertinencia.
4. **Conversión:** adquiere, se registra o solicita acceso según el producto.
5. **Activación:** comienza a usar el recurso o formación.
6. **Retención:** mantiene relación mediante valor continuo.
7. **Recomendación:** comparte o participa en afiliación si aplica.

## Tipos de embudo conceptual

- Embudo educativo.
- Embudo de curso o programa.
- Embudo de recurso descargable.
- Embudo de suscripción.
- Embudo de afiliados.
- Embudo institucional o B2B.

## Reglas

Cada embudo futuro deberá indicar objetivo, segmento, activo inicial, punto de conversión, métricas y restricciones de comunicación.
