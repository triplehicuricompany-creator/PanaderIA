# Ofertas y promociones

## Objetivo

Definir lineamientos para construir ofertas y promociones futuras sin redactar contenido comercial definitivo.

## Componentes permitidos a nivel conceptual

- Producto o paquete base.
- Público objetivo.
- Beneficio central a validar.
- Condición de acceso.
- Duración o vigencia propuesta.
- Bonos o complementos potenciales.
- Restricciones y términos pendientes.
- Indicadores de éxito.

## Tipos de oferta conceptual

- Lanzamiento.
- Paquete educativo.
- Acceso anticipado.
- Promoción temporal.
- Continuidad o renovación.
- Afiliado o alianza.
- Institucional.

## Reglas de control

- No prometer resultados no verificables.
- No usar urgencia falsa.
- No definir precios finales sin aprobación.
- No publicar condiciones incompletas.
- No alterar contenido del libro para justificar una oferta.
