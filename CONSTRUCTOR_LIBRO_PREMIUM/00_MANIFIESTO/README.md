# 00_MANIFIESTO

Define el alcance, reglas y criterios editoriales del Constructor del Libro Maestro Premium.

## Reglas maestras

- No modificar el Libro Maestro, Motor Maestro, Biblioteca Visual IA, Centro Multimedia ni recursos descargables existentes.
- Usar esta carpeta únicamente como capa de ensamblado, referencia y validación.
- Priorizar material real del Chef Juanote™ sobre recursos IA provisionales cuando ambos existan.
- Mantener a Boly™ como acompañante didáctico, no como autoridad técnica sustituta.
- Registrar cada recurso usado por capítulo antes de exportar.
