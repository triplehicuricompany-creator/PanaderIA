# Guía estándar para construir capítulos premium

Cada capítulo del Libro Maestro Premium debe ensamblarse siempre con esta estructura para asegurar consistencia editorial, didáctica y comercial.

## Estructura obligatoria

1. **Introducción**
   - Presenta el objetivo del capítulo, el resultado esperado y la importancia para el alumno.
2. **Explicación**
   - Desarrolla la teoría y técnica principal usando el Libro Maestro y los documentos base aprobados.
3. **Ilustraciones**
   - Integra imágenes, diagramas, infografías o secuencias paso a paso desde la Biblioteca Visual IA y recursos aprobados.
4. **Consejos de Boly™**
   - Agrega recordatorios claros, motivadores y prácticos para acompañar al alumno.
5. **Errores comunes**
   - Enumera fallas frecuentes, causa probable, señal visual y corrección recomendada.
6. **Video QR**
   - Vincula el video correspondiente mediante QR dinámico o ruta del Centro de Producción Multimedia.
7. **Práctica**
   - Define ejercicio aplicable, materiales requeridos, tiempo estimado y evidencia esperada.
8. **Checklist**
   - Incluye lista de verificación para confirmar dominio antes de avanzar.
9. **Resumen**
   - Sintetiza aprendizajes clave y criterios de calidad del capítulo.
10. **Autoevaluación**
    - Incluye preguntas, rúbrica breve o evidencia de desempeño para medir comprensión.

## Metadatos mínimos por capítulo

- Número de capítulo.
- Título oficial.
- Objetivo pedagógico.
- Fuentes base usadas.
- Recursos visuales vinculados.
- Videos vinculados.
- QR vinculados.
- Recursos descargables vinculados.
- Material Chef Juanote™ requerido o disponible.
- Consejos de Boly™ incluidos.
- Estado: `borrador`, `revision`, `aprobado`, `exportable`.

## Criterio de capítulo exportable

Un capítulo solo puede pasar a exportación si contiene las diez secciones obligatorias, sus recursos están registrados y no hay referencias rotas.
