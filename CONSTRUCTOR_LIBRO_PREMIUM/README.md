# Constructor del Libro Maestro Premium Beta 1.0

El **Constructor del Libro Maestro Premium de PanaderIA™** organiza el ensamblado automático de la versión comercial final del curso sin modificar los recursos existentes. Su función es actuar como capa de integración entre el Libro Maestro, el Motor Maestro, la Biblioteca Visual IA, el Centro de Producción Multimedia, los recursos descargables, Boly™ y el material del Chef Juanote™.

## Objetivo

Centralizar la ruta de construcción para convertir cada capítulo en una unidad premium consistente, exportable y lista para publicación comercial.

## Flujo completo de ensamblado

1. **Lectura del manifiesto**: revisar `manifest.json` para validar fuentes, carpetas, reglas, estado y estructura estándar.
2. **Preparación del capítulo**: usar `01_CAPITULOS/GUIA_ESTRUCTURA_CAPITULO.md` como molde obligatorio para cada capítulo.
3. **Vinculación de contenido base**: tomar el texto pedagógico desde el Libro Maestro, el Curso Maestro y los capítulos descargables existentes.
4. **Asignación de recursos**: relacionar imágenes, diagramas, infografías, videos, QR, recursos descargables, checklists y autoevaluaciones desde sus carpetas de origen.
5. **Integración Chef Juanote™**: incorporar demostraciones, notas técnicas y material real cuando exista, priorizando autoridad humana sobre recursos IA provisionales.
6. **Integración Boly™**: agregar consejos, advertencias, recordatorios y acompañamiento del alumno sin reemplazar la voz técnica del Chef Juanote™.
7. **Validación didáctica**: verificar que cada capítulo contenga introducción, explicación, ilustraciones, errores comunes, práctica, checklist, resumen y autoevaluación.
8. **Control de reemplazos**: registrar cuando un recurso IA provisional sea sustituido por fotografía, video o material real.
9. **Preexportación**: preparar la versión ensamblada en `10_EXPORTACION` con índice, capítulos, anexos y rutas de recursos.
10. **Exportación comercial**: generar el paquete final del Libro Maestro Premium para PDF, web, plataforma o distribución descargable.

## Carpetas del constructor

| Carpeta | Función |
|---|---|
| `00_MANIFIESTO` | Reglas editoriales, criterios de aprobación y alcance del constructor. |
| `01_CAPITULOS` | Plantilla y control de construcción de capítulos. |
| `02_RECURSOS` | Mapa de recursos descargables y anexos del alumno. |
| `03_IMAGENES` | Registro de imágenes, diagramas e ilustraciones usadas por capítulo. |
| `04_INFOGRAFIAS` | Integración de infografías educativas. |
| `05_VIDEOS` | Asociación de videos del Centro de Producción Multimedia. |
| `06_QR` | Control de QR dinámicos para videos, recursos y rutas web. |
| `07_BOLY` | Consejos, cápsulas y apoyos de Boly™ por capítulo. |
| `08_CHECKLISTS` | Listas de verificación por práctica, capítulo y exportación. |
| `09_AUTOEVALUACIONES` | Estructura de autoevaluaciones y evidencias de aprendizaje. |
| `10_EXPORTACION` | Salida ensamblada para versión comercial final. |

## Regla principal

Este constructor **no modifica recursos existentes**. Solo referencia, organiza, valida y ensambla materiales ya creados o aprobados dentro de la infraestructura de PanaderIA™.

## Resultado esperado

**Constructor del Libro Maestro Premium Beta 1.0** listo para ensamblar la versión comercial de PanaderIA™ con estructura homogénea, integración multimedia y control de calidad editorial.
