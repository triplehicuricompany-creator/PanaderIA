# Motor Maestro de Producción IA de PanaderIA™ — Beta 1.0

Estructura operativa para producir imágenes IA provisionales, diagramas, infografías, ilustraciones técnicas, miniaturas, banners y recursos por canal sin duplicar planes ni tocar el Libro Maestro.

## Carpetas principales
- `00_MANIFIESTO_MAESTRO/`: manifiesto global de recursos Beta.
- `01_IMAGENES_IA_PROVISIONALES/` a `10_PAQUETES_OFICIALES/`: bandejas por tipo/canal de recurso.
- `11_MATERIAL_REAL_CHEF_JUANOTE/`: recepción futura de fotografía, video y audio real.
- `12_REEMPLAZOS_Y_COMPLEMENTOS/`: bitácora de sustituciones y apoyos complementarios.
- `capitulos/`: 14 carpetas listas para producción Beta.

## Criterio operativo
Boly™ ayuda a ordenar, diagnosticar y explicar; Chef Juanote™ valida el estándar real. Todo recurso IA nace como provisional, reemplazable y conservable si aporta claridad pedagógica.
