# Producción Beta 1.0 — Capítulo 08: Corte profesional del bolillo

Carpeta de trabajo para producir recursos provisionales y organizar futuros materiales reales del Chef Juanote™.

## Objetivo pedagógico
apertura controlada y dirección correcta del corte.

## Regla de oro
No reescribir el Libro Maestro ni modificar capítulos: aquí solo se ordenan recursos Beta reemplazables y complementarios. Boly™ acompaña con claridad, cariño y diagnóstico práctico.

## Archivos
- `manifest.json`: inventario operativo por recurso.
- `prompts.md`: prompts base para IA.
- `recursos-provisionales.md`: lista de recursos Beta generables.
- `pendientes-chef-juanote.md`: material real prioritario.
- `reemplazos-futuros.md`: criterio para sustituir o complementar sin borrar IA útil.
