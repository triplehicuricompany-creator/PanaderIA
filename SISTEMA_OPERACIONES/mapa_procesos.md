# Mapa conceptual de procesos

| Proceso | Entrada | Salida esperada | Dependencias |
| --- | --- | --- | --- |
| Planeación operativa | Objetivos aprobados | Plan conceptual de ejecución futura | Gobernanza, calidad |
| Producción documental | Requerimiento autorizado | Documento revisable | Contenido educativo, legal |
| Revisión de calidad | Documento o flujo propuesto | Observaciones y estado | Sistema de calidad |
| Coordinación comercial | Necesidad de campaña o venta | Solicitud estructurada | Comercial, marketing, ventas |
| Gestión de recursos | Necesidad de material o activo | Registro de solicitud futura | Finanzas, banco de recursos |
| Seguimiento operativo | Indicador conceptual | Lectura no vinculante | Analítica, seguridad |
| Cierre documental | Evidencia de revisión | Archivo preparado para misión futura | Gobernanza |

## Uso permitido

El mapa sirve para visualizar dependencias, no para iniciar operación real.
