# Flujos de trabajo futuros

## Flujo base de solicitud

1. Identificar necesidad documental.
2. Confirmar sistema responsable.
3. Registrar alcance, exclusiones y dependencias.
4. Enviar a revisión humana.
5. Clasificar estado documental.
6. Programar misión futura si aplica.

## Flujo de revisión operativa

1. Revisar consistencia con sistemas existentes.
2. Detectar riesgos de duplicidad o contradicción.
3. Marcar validaciones especializadas pendientes.
4. Emitir recomendación documental.

## Flujo de cierre

Un flujo solo puede cerrarse como `APROBADO_PARA_MISION_FUTURA` cuando no requiere ejecución inmediata ni modifica documentos cerrados.
