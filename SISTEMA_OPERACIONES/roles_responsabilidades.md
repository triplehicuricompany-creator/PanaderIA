# Roles y responsabilidades conceptuales

| Rol funcional | Responsabilidad conceptual | Restricción |
| --- | --- | --- |
| Coordinación operativa | Ordenar prioridades y dependencias. | No asigna personas reales. |
| Revisión de calidad | Validar claridad, consistencia y trazabilidad. | No sustituye pruebas finales. |
| Revisión legal | Detectar implicaciones regulatorias o contractuales. | No emite asesoría jurídica definitiva. |
| Revisión financiera | Evaluar impacto presupuestal futuro. | No aprueba gasto real. |
| Revisión técnica | Identificar necesidades de herramientas o integraciones. | No implementa software. |
| Custodia documental | Mantener estructura y nomenclatura. | No altera misiones cerradas. |

## Regla principal

Los roles son funciones documentales. No representan cargos laborales, contratos ni responsables nominales.
