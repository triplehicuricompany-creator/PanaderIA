# Índices Maestros de Analítica

## Índice por dominio

- Libro: activos editoriales, capítulos, recursos y revisiones.
- Educación: avance, comprensión, práctica y certificación.
- Comercial: productos, ofertas, ventas, conversión y recurrencia.
- Clientes: perfiles, soporte, comunidad y fidelización.
- Marketing: canales, campañas, contenido y atribución conceptual.
- Multimedia: videos, audios, imágenes, QR y descargables.
- IA: asistentes, respuestas, calidad, seguridad y utilidad.

## Índice por tipo de medición

- Métricas generales.
- Indicadores operativos.
- KPIs estratégicos.
- Eventos conceptuales.
- Estadísticas descriptivas.
- Estadísticas comparativas.
- Señales de auditoría.
- Alertas de calidad.

## Índice por salida documental

- Dashboard conceptual ejecutivo.
- Dashboard conceptual educativo.
- Dashboard conceptual comercial.
- Dashboard conceptual clientes.
- Dashboard conceptual marketing.
- Dashboard conceptual multimedia.
- Dashboard conceptual IA.
- Reporte mensual conceptual.
- Reporte trimestral conceptual.
- Reporte de auditoría conceptual.
