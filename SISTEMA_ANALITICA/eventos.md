# Eventos Medibles Conceptuales

## Definición

Un evento medible conceptual describe una acción futura que podría registrarse si una fase posterior aprueba instrumentación técnica.

## Familias de eventos

- Lectura o consulta de contenido.
- Descarga de recurso.
- Inicio, avance o finalización de lección.
- Interacción con asistente IA.
- Visualización de video.
- Solicitud de soporte.
- Compra o intención comercial.
- Participación comunitaria.
- Escaneo de QR.

## Reglas

- Cada evento deberá tener propósito claro.
- No se deberán registrar datos sensibles innecesarios.
- No se aprobarán eventos que invadan privacidad o no aporten valor.
- Todo evento deberá mapearse a una métrica o reporte futuro.
