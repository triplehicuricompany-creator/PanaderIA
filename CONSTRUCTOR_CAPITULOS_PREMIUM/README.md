# Constructor Automático de Capítulos Premium

Sistema reutilizable para construir los capítulos 3 al 14 del **Libro Maestro Premium** con una estructura editorial uniforme, sin generar capítulos finales ni modificar capítulos existentes.

## Propósito

Este constructor define una metodología común para que cada capítulo premium mantenga el mismo nivel de claridad, profundidad técnica, tono didáctico y experiencia visual.

## Alcance

- Aplica a capítulos 3 al 14.
- No contiene capítulos desarrollados.
- No reemplaza contenido existente.
- Sirve como base editorial, técnica y de control de calidad.

## Estructura estándar obligatoria

Cada capítulo construido con este sistema debe incluir, en este orden:

1. Bienvenida del Chef Juanote.
2. Objetivo del capítulo.
3. Explicación técnica.
4. Consejos de Boly.
5. Errores comunes.
6. Ejercicio práctico.
7. Checklist.
8. Autoevaluación.
9. Recursos descargables.
10. Imágenes necesarias.
11. Videos.
12. QR.
13. Resumen final.

## Uso recomendado

1. Copiar `plantilla_capitulo.md` como base de trabajo temporal.
2. Sustituir los marcadores entre corchetes por información del capítulo correspondiente.
3. Completar las plantillas auxiliares según la necesidad del capítulo.
4. Validar el resultado con `plantilla_checklist.md`.
5. No guardar el resultado como capítulo final dentro de esta carpeta.

## Convenciones editoriales

- Mantener lenguaje claro, profesional y motivador.
- Usar al Chef Juanote como guía principal.
- Usar a Boly para consejos breves, memorables y prácticos.
- Priorizar aprendizaje aplicable en panadería real.
- Evitar duplicar secciones o alterar el orden estándar.

## Archivos incluidos

- `plantilla_capitulo.md`: estructura maestra del capítulo.
- `plantilla_checklist.md`: control de calidad editorial y técnico.
- `plantilla_autoevaluacion.md`: formato de evaluación para el alumno.
- `plantilla_consejos_boly.md`: guía para cápsulas de consejos.
- `plantilla_recursos_multimedia.md`: inventario de imágenes y videos.
- `plantilla_qr.md`: estructura para códigos QR.
- `plantilla_descargables.md`: guía de recursos descargables.
- `manifest.json`: metadatos del constructor.
