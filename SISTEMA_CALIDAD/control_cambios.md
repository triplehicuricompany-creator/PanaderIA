# Control de Cambios Aprobados

## Propósito

Registrar los cambios aprobados por el Sistema Maestro de Calidad PanaderIA™ para mantener trazabilidad antes de futuras publicaciones.

## Reglas de registro

1. Todo cambio aprobado debe tener fecha, ID, descripción, tipo de revisión y responsable.
2. No se registran como aprobados los elementos con bloqueos críticos.
3. Los cambios observados deben permanecer fuera del registro de aprobación hasta corregirse.
4. Cada publicación debe poder rastrearse hasta una revisión final.
5. Este archivo registra aprobaciones; no contiene contenido final del libro.
