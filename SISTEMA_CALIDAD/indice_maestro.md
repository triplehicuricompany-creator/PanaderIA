# Índice Maestro — Sistema Maestro de Calidad PanaderIA™

## Función

Centralizar la navegación del sistema y permitir que cada revisión de calidad encuentre su documento rector sin depender de procesos externos.

## Bloques documentales

1. **Gobierno y estructura**: `arquitectura_documental.md`, `gobernanza.md`, `politicas.md`.
2. **Marco rector**: `documentacion_maestra.md`, `estandares_calidad.md`.
3. **Calidad operativa conceptual**: `control_calidad.md`, `aseguramiento_calidad.md`.
4. **Validaciones especializadas**: `revision_editorial.md`, `validacion_tecnica.md`, `validacion_pedagogica.md`, `validacion_multimedia.md`.
5. **Aceptación y seguimiento**: `criterios_aceptacion.md`, `gestion_incidencias.md`, `mejora_continua.md`.
6. **Trazabilidad y evolución**: `versionado_documental.md`, `auditorias_calidad.md`, `roadmap.md`.

## Orden recomendado de consulta

1. Leer `README.md` y `manifest.json`.
2. Confirmar alcance en `documentacion_maestra.md`.
3. Aplicar estándares desde `estandares_calidad.md`.
4. Seleccionar validaciones especializadas según el tipo de activo.
5. Resolver criterios de aceptación, incidencias y versionado.
6. Registrar aprendizajes para mejora continua y auditorías futuras.
