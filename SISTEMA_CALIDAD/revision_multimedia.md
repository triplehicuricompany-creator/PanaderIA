# Revisión Multimedia

## Objetivo

Establecer el protocolo para validar imágenes, videos, audios, gráficos y otros activos multimedia vinculados a futuras integraciones.

## Puntos de revisión

- Relación directa con el contenido o aprendizaje que acompaña.
- Calidad visual o sonora suficiente para uso premium.
- Nomenclatura, versión y ubicación documentadas.
- Formato adecuado para publicación y respaldo.
- Derechos de uso, autoría o licencia registrados cuando aplique.
- Accesibilidad mediante texto alternativo, título o descripción.
- Coherencia de marca, estilo visual y nivel técnico.
- Ausencia de errores visuales que contradigan la enseñanza.

## Validación por tipo

| Tipo | Validación mínima |
|---|---|
| Imagen | Resolución, enfoque, pertinencia y texto alternativo. |
| Video | Audio claro, duración justificada, portada y enlace correcto. |
| Audio | Limpieza sonora, volumen estable y propósito pedagógico. |
| Gráfico | Legibilidad, escala, fuente y consistencia visual. |
