# Control de Calidad

## Definición

El control de calidad es la revisión puntual de un activo para detectar errores, desviaciones o incumplimientos antes de su aprobación.

## Enfoque

- Comparar el activo contra criterios definidos.
- Identificar defectos editoriales, técnicos, pedagógicos, multimedia o documentales.
- Clasificar hallazgos según severidad e impacto.
- Determinar si el activo se acepta, observa o rechaza.

## Salidas conceptuales

- Lista de hallazgos.
- Estado recomendado.
- Criterios incumplidos.
- Acciones sugeridas.
- Evidencia documental de revisión.
