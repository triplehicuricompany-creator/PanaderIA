# Criterios Oficiales de Calidad

## Propósito

Definir los criterios mínimos y premium que toda integración futura debe cumplir antes de considerarse aprobada para el Libro Maestro Premium PanaderIA™.

## Criterios generales

1. **Precisión técnica:** la información debe ser correcta, verificable y coherente con buenas prácticas panaderas.
2. **Claridad pedagógica:** cada recurso debe ser comprensible para su audiencia definida.
3. **Coherencia editorial:** tono, estructura, vocabulario y marca deben mantenerse consistentes.
4. **Utilidad práctica:** el contenido o recurso debe ayudar a aprender, ejecutar, revisar o profundizar.
5. **Trazabilidad:** todo elemento debe tener estado, versión, fecha y responsable de revisión.
6. **Accesibilidad:** materiales visuales, enlaces y descargables deben favorecer lectura y uso claros.
7. **No duplicación:** no se debe repetir material sin valor agregado comprobable.
8. **Listo para publicación:** ningún recurso debe depender de instrucciones ambiguas o pendientes críticos.

## Niveles de evaluación

| Nivel | Descripción | Acción |
|---|---|---|
| Crítico | Error técnico, legal, pedagógico o de publicación que impide uso seguro. | Bloquear publicación. |
| Mayor | Inconsistencia importante que afecta comprensión, calidad o experiencia. | Corregir antes de aprobar. |
| Menor | Ajuste de forma, formato o claridad sin impacto crítico. | Corregir o registrar como pendiente no bloqueante. |
| Aprobado | Cumple criterios y no tiene pendientes bloqueantes. | Autorizar integración. |

## Evidencia mínima

Cada revisión debe registrar:

- Elemento revisado.
- Fecha de revisión.
- Tipo de revisión aplicada.
- Resultado.
- Observaciones.
- Responsable o rol aprobador.
