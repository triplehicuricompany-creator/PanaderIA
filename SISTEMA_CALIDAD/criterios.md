# Criterios Generales de Calidad

Este documento se conserva como puente documental con versiones previas del Sistema Maestro de Calidad.

Los criterios oficiales vigentes se organizan en:

- `estandares_calidad.md` para estándares generales.
- `criterios_aceptacion.md` para aceptación, observación y rechazo.
- `control_calidad.md` para revisión puntual de activos.
- `aseguramiento_calidad.md` para prevención de desviaciones.

## Criterio rector

Todo activo debe ser correcto, claro, útil, coherente, trazable y alineado con las restricciones oficiales del ecosistema PanaderIA™.
