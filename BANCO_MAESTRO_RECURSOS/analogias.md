# Analogías pedagógicas

## Propósito

Reunir comparaciones didácticas para explicar procesos panaderos de forma accesible. Deben adaptarse al nivel técnico del lector.

## Masa y gluten

- El gluten como una red elástica que sostiene el gas.
- La masa como una tela: si no se trabaja bien, se rompe; si se trabaja de más, se fatiga.
- El amasado como entrenamiento gradual, no como castigo.

## Fermentación

- La fermentación como una conversación lenta entre levadura, tiempo y temperatura.
- La masa como un globo vivo: necesita presión, pero también límites.
- El reposo como una pausa estratégica, no como tiempo perdido.

## Horno

- El horno como escenario final donde se revela lo hecho antes.
- El vapor como la ventana de oportunidad para expansión y brillo.
- La corteza como armadura comestible que protege la miga.

## Negocio

- La receta como promesa de marca.
- La vitrina como vendedor silencioso.
- La consistencia como contrato invisible con el cliente.
