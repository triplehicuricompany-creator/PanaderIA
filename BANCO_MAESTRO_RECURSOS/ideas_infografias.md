# Ideas de infografías

## Propósito

Clasificar ideas visuales reutilizables para diagramas, tablas, árboles de decisión e infografías.

## Infografías técnicas

- Ciclo completo del bolillo desde pesado hasta enfriado.
- Relación entre temperatura, tiempo y fermentación.
- Mapa visual de defectos: volumen, greña, miga, corteza y color.
- Comparativa de masas subamasadas, correctas y sobreamasadas.

## Diagramas de diagnóstico

- Árbol de decisión para bajo volumen.
- Árbol de decisión para greña cerrada.
- Flujo para detectar fermentación corta o excesiva.
- Matriz causa-solución para corteza pálida u oscura.

## Infografías comerciales

- Estructura de costos por pieza.
- Diseño básico de vitrina rentable.
- Relación entre producción, merma y margen.
