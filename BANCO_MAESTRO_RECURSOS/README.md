# BANCO_MAESTRO_RECURSOS

Biblioteca oficial de recursos reutilizables del proyecto PanaderIA™.

## Propósito

Este banco centraliza materiales de apoyo, referencias internas, ideas, clasificaciones y recursos reutilizables para futuras versiones del libro y sus extensiones digitales.

## Alcance

- No contiene capítulos terminados.
- No sustituye al libro maestro ni a los módulos existentes.
- No modifica el contenido narrativo final.
- Sirve como fuente consultable para cualquier capítulo, curso, recurso multimedia, descargable o asistente IA del ecosistema PanaderIA™.

## Uso recomendado

1. Consultar `manifest.json` para identificar el archivo adecuado.
2. Extraer únicamente recursos reutilizables.
3. Adaptar cada recurso al capítulo, formato o medio correspondiente.
4. Mantener trazabilidad cuando un recurso se convierta en material final.

## Índice

- `frases_chef_juanote.md`: frases reutilizables para Chef Juanote.
- `frases_boly.md`: frases reutilizables para Boly.
- `errores_comunes.md`: clasificación de fallas frecuentes.
- `soluciones_panaderia.md`: banco de soluciones y criterios de intervención.
- `analogias.md`: analogías pedagógicas reutilizables.
- `glosario.md`: términos base del ecosistema panadero.
- `tips_produccion.md`: consejos operativos para producción.
- `tips_comerciales.md`: consejos comerciales y de venta.
- `ideas_multimedia.md`: ideas generales para recursos multimedia.
- `ideas_infografias.md`: ideas para infografías y diagramas.
- `ideas_videos.md`: ideas para videos educativos.
- `ideas_descargables.md`: ideas para hojas, plantillas y checklists.

## Criterio editorial

Todo recurso debe mantenerse como material de apoyo, referencia o idea reutilizable. La redacción final de capítulos, guiones completos o piezas editoriales terminadas debe realizarse fuera de este banco.
