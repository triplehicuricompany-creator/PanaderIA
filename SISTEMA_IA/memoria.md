# Memoria conceptual de IA

## Objetivo

Definir límites y criterios para cualquier memoria futura asociada a asistentes PanaderIA™.

## Tipos de memoria conceptuales

### Memoria de sesión

Contexto temporal de una conversación activa, sin persistencia documental obligatoria.

### Memoria de usuario

Preferencias o progreso de una persona, solo si existe autorización, política de privacidad y control de eliminación.

### Memoria institucional

Conocimiento oficial del ecosistema, derivado de sistemas maestros aprobados.

### Memoria operativa

Registros de tareas, estados o revisiones internas, sujetos a trazabilidad y permisos.

## Reglas

- No recordar datos sensibles sin finalidad clara.
- No usar memoria para evadir límites de actuación.
- No mezclar preferencias del usuario con conocimiento canónico.
- Toda memoria persistente futura deberá tener mecanismo de revisión, corrección y eliminación.

## Estado actual

No se implementa memoria funcional en esta misión.
