# Asistentes futuros especializados

## Objetivo

Definir lineamientos para futuros asistentes de Pastelería, Chocolatería y nuevas áreas del ecosistema PanaderIA™.

## Asistente de Pastelería

Podrá orientar en conceptos, procesos, organización de aprendizaje y acompañamiento práctico de pastelería únicamente cuando exista base de conocimiento aprobada para esa disciplina.

## Asistente de Chocolatería

Podrá apoyar consultas sobre chocolatería, procesos, vocabulario y práctica guiada cuando exista documentación canónica validada.

## Requisitos previos para crear un asistente

- Sistema o módulo de conocimiento aprobado.
- Personalidad documentada.
- Límites de actuación definidos.
- Prompts maestros controlados.
- Flujo de conversación validado.
- Política de escalamiento humano.
- Relación clara con Boly™ y Chef Juanote™.

## Prohibición

No se crean asistentes nuevos en esta misión. Solo se documentan criterios para su creación futura.
