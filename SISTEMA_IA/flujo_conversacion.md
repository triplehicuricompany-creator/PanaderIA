# Flujo de conversación

## Objetivo

Diseñar el flujo conceptual de interacción de asistentes IA sin generar conversaciones finales.

## Flujo base

1. Recepción de intención del usuario.
2. Identificación de dominio.
3. Verificación de alcance del asistente.
4. Consulta conceptual a fuentes autorizadas.
5. Respuesta clara y limitada.
6. Señalamiento de incertidumbre cuando aplique.
7. Escalamiento humano si el caso lo requiere.
8. Cierre responsable o propuesta de siguiente paso.

## Tipos de intención

- Aprender un concepto.
- Resolver una duda práctica.
- Diagnosticar un error común.
- Repasar un capítulo.
- Solicitar material o recurso.
- Pedir recomendación operativa.
- Reportar problema técnico.
- Solicitar acción fuera de alcance.

## Escalamiento obligatorio

Debe escalarse cuando exista riesgo de seguridad, datos sensibles, decisión comercial crítica, modificación de contenido oficial, certificación, diagnóstico no verificable o contradicción entre fuentes.
