# Prompts maestros

## Objetivo

Definir la estructura de control para prompts maestros de PanaderIA™ sin generar conversaciones finales ni prompts ejecutables de producción.

## Tipos de prompts futuros

- Prompt de identidad del asistente.
- Prompt de tono y personalidad.
- Prompt de consulta al conocimiento.
- Prompt de diagnóstico orientativo.
- Prompt de práctica educativa.
- Prompt de revisión documental.
- Prompt de seguridad y límites.
- Prompt de escalamiento humano.

## Estructura mínima

Todo prompt maestro futuro deberá incluir:

- Identificador.
- Propósito.
- Asistente asociado.
- Entradas permitidas.
- Salidas esperadas.
- Fuentes autorizadas.
- Restricciones.
- Casos de rechazo.
- Criterios de evaluación.
- Versión.

## Reglas

- No usar prompts sin versión.
- No mezclar personalidad con permisos técnicos.
- No permitir que un prompt contradiga políticas o límites.
- No publicar prompts sensibles si contienen lógica de seguridad, claves o instrucciones internas restringidas.
