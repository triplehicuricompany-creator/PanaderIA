# Dependencias entre Sistemas Maestros

## Propósito

Registrar cómo se relacionan conceptualmente los Sistemas Maestros del ecosistema PanaderIA™ para evitar contradicciones, duplicidad y crecimiento desordenado.

## Tipos de dependencia

### Dependencia editorial

Cuando un sistema requiere coherencia con estructura, capítulos, referencias o criterios narrativos.

### Dependencia pedagógica

Cuando un sistema depende de objetivos de aprendizaje, evaluaciones, prácticas o progresión formativa.

### Dependencia técnica

Cuando un sistema se relaciona con IA, automatizaciones, analítica, integraciones, plataforma o seguridad.

### Dependencia multimedia

Cuando un sistema requiere coordinación con guías de video, fotografía, audio, música, QR o producción audiovisual.

### Dependencia de marca

Cuando un sistema se vincula con identidad, comunicación, marketing o lanzamiento.

## Reglas de dependencia

- Toda dependencia transversal debe documentarse antes de cambios mayores.
- Ningún sistema debe duplicar funciones centrales de otro sistema.
- Las dependencias deben respetar el alcance original de cada sistema.
- La gobernanza debe actuar como punto de referencia para resolver conflictos de alcance.

## Mapa conceptual inicial

- Libro, curso y descargables comparten dependencias editoriales y pedagógicas.
- IA, automatizaciones y analítica comparten dependencias técnicas y de trazabilidad.
- Marketing e identidad de marca dependen de criterios estratégicos y de comunicación.
- Producción multimedia se relaciona con contenidos audiovisuales, recursos visuales y soportes de aprendizaje.
- Gobernanza coordina criterios comunes sin reemplazar a los sistemas especializados.
