# Catálogo Maestro de Fotografías de PanaderIA™

**Documento:** Guía oficial de producción fotográfica para PanaderIA™.
**Alcance:** Libro Maestro de Bolillo Premium, Curso Maestro de Bolillo Premium, Plataforma PanaderIA™, recursos descargables, web, redes sociales y publicidad.
**Criterio editorial:** este catálogo no agrega capítulos nuevos ni modifica contenido técnico; organiza fotografías necesarias a partir de los capítulos, recursos visuales y guion multimedia existentes.

---

## Reglas maestras de fotografía

- Toda fotografía debe enseñar una decisión técnica, una señal visual o un criterio de calidad.
- Las manos, herramientas, ingredientes y errores deben ser reales; evitar renders cuando el objetivo sea diagnóstico.
- Cada capítulo debe tener mínimo: foto obligatoria, secuencia paso a paso, error común, resultado correcto, comparativa y foto con Chef Juanote/Boly™.
- Boly™ debe aparecer como apoyo visual, no como sustituto del trabajo real del Chef Juanote.
- Las fotos para descargables deben tener espacio limpio para anotaciones, flechas, etiquetas o QR.
- Las fotos para publicidad deben priorizar apetito, confianza, claridad y acabado profesional.
- Las fotos comparativas deben mantener el mismo fondo, ángulo, distancia e iluminación para que la diferencia sea justa.
- Nombrar archivos con el código sugerido: C##_F##_descripcion_uso_v01.ext.

## Resumen numérico

- **Capítulos cubiertos:** 14.
- **Fotografías mínimas por capítulo:** 6.
- **Total maestro mínimo:** 84 fotografías.
- **Críticas para lanzamiento:** 70 fotografías — las cinco de prioridad alta por cada capítulo: obligatoria, paso a paso, errores comunes, resultado correcto y comparativa, concentrando primero capítulos de prioridad alta.
- **Producción diaria aprovechable:** 14 fotografías — las de Chef Juanote/Boly™ pueden capturarse durante prácticas reales del obrador; además, varias tomas de resultado y comparativa pueden aprovechar producción diaria si se planifican con luz y fondo constantes.
