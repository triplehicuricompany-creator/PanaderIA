# Ingresos Conceptuales

## Categorías futuras

| Categoría | Descripción | Relación documental |
|---|---|---|
| Libro premium | Venta futura del libro o paquetes editoriales. | `SISTEMA_PUBLICACION`, `SISTEMA_COMERCIAL` |
| Curso premium | Acceso educativo futuro. | `SISTEMA_CURSO`, `SISTEMA_VENTAS` |
| Membresías | Acceso recurrente futuro. | `SISTEMA_VENTAS` |
| Licencias | Uso institucional o empresarial futuro. | `SISTEMA_LEGAL`, `SISTEMA_CLIENTES` |
| Afiliados | Modelo comercial futuro no activado. | `SISTEMA_MARKETING` |
| Servicios complementarios | Soporte, asesorías o talleres futuros. | Pendiente de definición |

## Reglas

- Ninguna categoría representa ingreso real.
- Ninguna proyección debe publicarse como promesa comercial.
- Todo ingreso futuro requiere trazabilidad, validación fiscal y sistema de registro autorizado.
