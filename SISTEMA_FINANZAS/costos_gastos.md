# Costos y Gastos

## Clasificación conceptual

| Tipo | Descripción | Ejemplos no operativos |
|---|---|---|
| Costo directo | Asociado a un producto o entrega específica. | diseño de recurso, edición de video, revisión técnica |
| Gasto operativo | Necesario para sostener el ecosistema. | herramientas, administración, soporte |
| Gasto comercial | Relacionado con adquisición y conversión. | anuncios futuros, materiales promocionales |
| Gasto legal/fiscal | Relacionado con cumplimiento. | asesoría, permisos, revisiones |
| Inversión | Mejora de capacidad futura. | plataforma, infraestructura, automatización |

## Campos futuros recomendados

- Categoría.
- Descripción.
- Área relacionada.
- Periodo.
- Responsable.
- Estado de validación.
- Evidencia o fuente.

## Límite

Este documento no contiene cifras reales ni autoriza gastos.
