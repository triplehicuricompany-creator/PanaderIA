# Reportes Financieros Futuros

## Tipos de reporte conceptual

| Reporte | Uso futuro | Requiere validación |
|---|---|---|
| Presupuesto vs. ejecución | Comparar planeación con uso real de recursos. | Sí |
| Ingresos por línea | Observar categorías comerciales. | Sí |
| Costos por proyecto | Entender costo de producción y operación. | Sí |
| Margen por producto | Evaluar rentabilidad. | Sí |
| Flujo de caja | Monitorear liquidez. | Sí |
| Riesgos financieros | Priorizar alertas y mitigaciones. | Sí |

## Campos mínimos futuros

- Periodo.
- Fuente de datos.
- Responsable.
- Estado de validación.
- Supuestos.
- Limitaciones.
- Próxima acción.

## Límite

Los reportes descritos no existen como archivos operativos ni se alimentan con datos reales durante esta misión.
