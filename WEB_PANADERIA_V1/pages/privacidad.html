Política de privacidad.

Documento base para informar cómo PanaderIA™ tratará datos personales cuando active formularios, tienda, academia, certificación e IA.

Datos que podrían recopilarse

  • Nombre, correo y datos de contacto.
  • Historial de compra, acceso y progreso académico.
  • Evidencias enviadas para certificación.
  • Mensajes enviados a soporte o Boly™ cuando se active la IA.

Uso de la información

Los datos se utilizarán para gestionar cursos, entregar productos, emitir certificados, responder solicitudes, mejorar recursos educativos y cumplir obligaciones legales.

Derechos

El alumno podrá solicitar acceso, corrección o eliminación de sus datos por los canales oficiales de contacto.
