# WEB_PANADERIA_V1

Primera versión estática de la página web oficial de PanaderIA™.

## Estructura

- `index.html`: página de inicio.
- `pages/`: páginas internas principales.
- `assets/styles.css`: identidad visual, layout responsive y componentes.
- `assets/main.js`: navegación móvil, acordeones FAQ, año dinámico y utilidades.

## Cómo revisar localmente

Abrir `index.html` directamente en el navegador o servir la carpeta con:

```bash
python3 -m http.server 8080 --directory WEB_PANADERIA_V1
```

Luego visitar `http://localhost:8080`.
