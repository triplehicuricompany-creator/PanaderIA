# Privacidad y Datos en Soporte

## Principios

1. Solicitar solo datos necesarios para resolver el caso.
2. Evitar recopilar contraseñas, datos bancarios completos o información sensible innecesaria.
3. Separar consultas educativas de datos comerciales o personales.
4. Escalar incidentes de seguridad a los sistemas correspondientes.
5. Definir retención, acceso y eliminación solo mediante misión futura validada legalmente.

## Estado

Este documento no constituye política de privacidad final ni aviso legal. Requiere revisión profesional antes de cualquier uso operativo.
