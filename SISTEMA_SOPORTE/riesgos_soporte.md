# Riesgos de Soporte

| Riesgo | Descripción | Mitigación conceptual |
|---|---|---|
| Promesas indebidas | Responder con garantías no autorizadas. | Plantillas revisadas y escalamiento. |
| Exposición de datos | Solicitar o almacenar información sensible. | Mínima información y validación legal. |
| Respuestas contradictorias | Soporte contradice ventas, legal o contenido. | Base de conocimiento versionada. |
| Automatización prematura | Bot responde casos delicados sin revisión. | Prohibir automatizaciones en esta misión. |
| Falta de trazabilidad | No existe historial suficiente para mejorar. | Definir modelo de registro futuro. |

## Nota

Los riesgos no activan controles operativos reales. Sirven para planear misiones futuras.
