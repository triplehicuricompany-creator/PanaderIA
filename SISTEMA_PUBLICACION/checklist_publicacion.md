# Checklist maestro antes de publicar

## Identificación

- [ ] Título final confirmado.
- [ ] Edición y versión registradas.
- [ ] Responsable de aprobación definido.
- [ ] Fecha objetivo documentada.
- [ ] Plataforma destino definida.

## Contenido

- [ ] Todos los capítulos están completos y aprobados.
- [ ] No existen marcas de borrador, comentarios internos ni pendientes visibles.
- [ ] El índice coincide con el contenido real.
- [ ] Las referencias cruzadas funcionan.
- [ ] Los anexos y recursos prometidos existen.

## Calidad técnica y editorial

- [ ] Ortografía y estilo revisados.
- [ ] Terminología panadera consistente.
- [ ] Instrucciones técnicas validadas.
- [ ] Tablas, fórmulas y porcentajes revisados.
- [ ] Imágenes, diagramas y capturas legibles.

## Recursos digitales

- [ ] Enlaces probados.
- [ ] QR probados.
- [ ] Archivos descargables organizados.
- [ ] Rutas locales eliminadas.
- [ ] Tamaño de archivo aceptable para la plataforma.

## Impresión

- [ ] Márgenes, sangrado y lomo definidos.
- [ ] Portada, contraportada y lomo revisados.
- [ ] ISBN o código interno definido si aplica.
- [ ] Prueba física solicitada o aprobada.
- [ ] Costos, regalías y precio revisados.

## Plataformas

- [ ] Metadatos completos.
- [ ] Descripción comercial revisada.
- [ ] Categorías y palabras clave definidas.
- [ ] Precio y moneda confirmados.
- [ ] Política de reembolso o soporte publicada.

## Aprobación final

- [ ] Control editorial aprobado.
- [ ] Control técnico aprobado.
- [ ] Control visual aprobado.
- [ ] Control legal/marca aprobado.
- [ ] Control comercial aprobado.
- [ ] Publicación autorizada por responsable final.

## Regla de cierre

Si una casilla crítica no está aprobada, el libro no debe publicarse. La publicación solo puede avanzar cuando la versión, los archivos, la plataforma y la oferta comercial coincidan exactamente.
