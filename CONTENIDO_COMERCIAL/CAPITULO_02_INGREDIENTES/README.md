# Capítulo 2 — Ingredientes del bolillo y función de cada uno

**Producto:** Libro Maestro Premium PanaderIA™  
**Versión:** Comercial Beta 1.0  
**Estado:** Listo para revisión del Chef Juanote  
**Personajes guía:** Chef Juanote y Boly  
**Nivel:** Principiantes

## Objetivo del capítulo

Ayudar al alumno a comprender qué aporta cada ingrediente del bolillo —harina, agua, levadura, sal y mejorantes opcionales— para que deje de ver la receta como una lista y empiece a tomar decisiones con criterio panadero.

## Archivos incluidos

- `capitulo_02_texto_premium.md`: texto principal desarrollado del capítulo.
- `checklist_capitulo_02.md`: lista de verificación para seleccionar y preparar ingredientes.
- `autoevaluacion_capitulo_02.md`: preguntas de repaso y reflexión.
- `guion_video_capitulo_02.md`: guion para video corto complementario.
- `prompts_imagenes_capitulo_02.md`: indicaciones y prompts para imágenes.
- `qr_pendiente_capitulo_02.md`: ficha de control para futuro código QR.

## Uso recomendado

1. Revisar el texto principal con enfoque técnico y comercial.
2. Validar que las explicaciones sean claras para principiantes.
