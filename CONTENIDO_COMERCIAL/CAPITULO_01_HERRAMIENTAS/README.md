# Capítulo 1 — Herramientas necesarias para comenzar a hacer bolillo

**Producto:** Libro Maestro Premium PanaderIA™  
**Versión:** Comercial Beta 1.0  
**Estado:** Listo para revisión del Chef Juanote  
**Personajes guía:** Chef Juanote y Boly  
**Nivel:** Principiantes

## Objetivo del capítulo

Ayudar al alumno a identificar, entender y preparar las herramientas básicas necesarias para comenzar a elaborar bolillo en casa o en un espacio de producción inicial, sin complicaciones y con criterios profesionales.

## Archivos incluidos

- `capitulo_01_texto_premium.md`: texto principal desarrollado del capítulo.
- `checklist_capitulo_01.md`: lista de verificación para el alumno.
- `autoevaluacion_capitulo_01.md`: preguntas de repaso y reflexión.
- `guion_video_capitulo_01.md`: guion para video corto complementario.
- `prompts_imagenes_capitulo_01.md`: indicaciones y prompts para imágenes.
- `qr_pendiente_capitulo_01.md`: ficha de control para futuro código QR.

## Uso recomendado

1. Revisar el texto principal.
2. Validar el tono comercial, educativo y mexicano.
3. Producir o seleccionar imágenes de apoyo.
4. Grabar el video corto sugerido.
5. Generar el QR cuando el recurso multimedia esté publicado.

## Nota editorial

Este capítulo está escrito con la voz cercana, profesional y motivadora del Chef Juanote. Boly participa como asistente amigable con consejos prácticos para principiantes.
