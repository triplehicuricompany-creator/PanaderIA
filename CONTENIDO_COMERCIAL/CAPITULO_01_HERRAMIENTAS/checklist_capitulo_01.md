# Checklist del alumno — Capítulo 1

**Tema:** Herramientas necesarias para comenzar a hacer bolillo  
**Nivel:** Principiantes

## Antes de iniciar

Marca cada punto cuando esté listo:

- [ ] Mi mesa o superficie de trabajo está limpia y despejada.
- [ ] Tengo una balanza digital o una forma confiable de pesar ingredientes.
- [ ] Verifiqué que la balanza funcione y pueda medir en gramos.
- [ ] Sé usar la función de tara de la balanza.
- [ ] Tengo una taza medidora para líquidos.
- [ ] Tengo un bowl o vasija grande, limpio y sin olores fuertes.
- [ ] Tengo batidora con gancho, amasadora o espacio suficiente para amasar a mano.
- [ ] Tengo una espátula o raspa de panadero.
- [ ] Tengo una charola limpia para hornear.
- [ ] Tengo trapo limpio o plástico de uso alimenticio para cubrir la masa.
- [ ] Tengo cuchillo filoso, navaja panadera o cutter limpio para hacer cortes.
- [ ] Tengo una rejilla o espacio ventilado para enfriar el pan.
- [ ] Revisé que mi levadura y mis ingredientes estén vigentes.
- [ ] Separé mis herramientas antes de comenzar.
- [ ] Tengo una libreta o archivo para anotar mis resultados.
