# Checklist — Capítulo 05: Fermentación

**Uso:** recurso imprimible para el alumno.  
**QR sugerido:** `QR-LMBP-C05-FERMENTACION`

## Antes de empezar

- [ ] Mesa limpia y despejada.
- [ ] Herramientas listas y limpias.
- [ ] Fórmula o ficha de trabajo visible.
- [ ] Tiempo disponible sin interrupciones.
- [ ] Registro de práctica preparado.

## Verificación técnica

- [ ] Entiendo el objetivo de fermentación.
- [ ] Identifiqué la señal principal que debo observar.
- [ ] Evité corregir varias variables al mismo tiempo.
- [ ] Fermentar poco.
- [ ] Fermentar de más.
- [ ] No cubrir la masa.
- [ ] Ignorar temperatura ambiente.

## Evidencia de práctica

- [ ] Tomé fotografía de evidencia.
- [ ] Registré tiempos y observaciones.
- [ ] Anoté un error posible y su corrección.
- [ ] Guardé la ficha para comparar con el resultado final.
