# Contenido comercial — Capítulo 03: Ingredientes del bolillo

Carpeta de recursos finales de usuario para producción comercial del Capítulo 03 del Curso Maestro de Bolillo.

## Archivos incluidos

- `capitulo_03_texto_premium.md`: capítulo premium base para el paquete final.
- `checklist_capitulo_03.md`: checklist imprimible de mise en place de ingredientes.
- `autoevaluacion_capitulo_03.md`: preguntas de comprensión antes de avanzar a amasado.
- `guion_video_capitulo_03.md`: guion de video demostrativo.
- `prompts_imagenes_capitulo_03.md`: prompts para generar o producir activos visuales.
- `qr/textos_qr_capitulo_03.md`: textos finales para QR de video, evidencia y diagnóstico.
- `ilustraciones_finales/`: ilustraciones SVG finales listas para maquetación.
- `prompts_finales/`: prompts definitivos para generación de imágenes.
- `boly_3d/`: prompts finales para render 3D de Boly™.
- `video/`: storyboard, guion completo, narración y subtítulos SRT.
- `imprimibles/`: checklist final y HTML fuente imprimible; el PDF final se generará posteriormente desde estos archivos fuente.
- `epub/fuente_epub/`: archivos fuente compatibles para generar el EPUB final posteriormente (XHTML, OPF, XML y mimetype en TXT).
- `descargables/`: recursos descargables del alumno.
- `materiales_alumnos/`: fichas de práctica listas para uso.

## Uso recomendado

Usar estos archivos como paquete comercial final del alumno y conectar las URLs definitivas de QR en la plataforma de distribución.


## Nota de generación de binarios

Los archivos binarios finales (`.pdf`, `.epub` u otros paquetes compilados) no se almacenan en esta rama. El PDF final debe generarse posteriormente a partir de `imprimibles/capitulo_03_imprimible.html` y `imprimibles/checklist_imprimible_final_capitulo_03.md`. El EPUB final debe generarse posteriormente a partir de los archivos fuente ubicados en `epub/fuente_epub/`.
