# Checklist — Capítulo 03: Ingredientes del bolillo

**Uso:** recurso imprimible para el alumno antes de iniciar amasado.  
**QR sugerido:** `QR-LMBP-C03-INGREDIENTES`

## Antes de empezar

- [ ] Mesa limpia y despejada.
- [ ] Báscula estable y en cero.
- [ ] Fórmula o ficha de trabajo visible.
- [ ] Recipientes limpios para cada ingrediente.
- [ ] Harina identificada por marca, tipo o lote si aplica.
- [ ] Agua medida y sin temperatura excesiva.
- [ ] Levadura pesada y separada.
- [ ] Sal pesada, visible y no olvidada.

## Verificación técnica

- [ ] Sé qué función cumple la harina.
- [ ] Sé qué función cumple el agua.
- [ ] Sé qué función cumple la levadura.
- [ ] Sé qué función cumple la sal.
- [ ] Revisé si el ambiente está muy frío o muy caliente.
- [ ] Evité agregar levadura extra para acelerar sin control.
- [ ] Evité modificar varios factores al mismo tiempo.

## Evidencia de práctica

- [ ] Tomé fotografía de la mesa con ingredientes pesados.
- [ ] Registré pesos usados.
- [ ] Registré observaciones de harina y ambiente.
- [ ] Anoté un posible riesgo antes de amasar.
- [ ] Guardé la ficha para comparar con el resultado final.
