# Roadmap de Versiones

Plan orientativo para la evolución controlada de PanaderIA™.

## Fase Beta

- Consolidar inventario de activos versionables.
- Validar correspondencia entre Libro Maestro, curso, multimedia y asistentes IA.
- Documentar dependencias entre versiones.

## Fase Release Candidate

- Revisar consistencia de registros.
- Confirmar que no existan pendientes críticos para publicación.
- Preparar manifiestos finales por área.

## Fase Final

- Declarar versión estable.
- Congelar referencias históricas.
- Abrir una nueva línea de trabajo para mejoras posteriores.

## Evolución futura

- Versionado específico de Boly y asistentes IA.
- Matriz de compatibilidad entre libro, curso y recursos multimedia.
- Archivo de versiones finales preservadas.
