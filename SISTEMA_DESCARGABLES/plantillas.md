# Plantillas Descargables

## Definición

Las plantillas son documentos reutilizables que el alumno, instructor o equipo editorial puede completar, adaptar o duplicar para organizar información.

## Tipos de plantillas

- Plantilla de receta estándar.
- Plantilla de fórmula panadera.
- Plantilla de plan de producción.
- Plantilla de costos.
- Plantilla de bitácora.
- Plantilla de calendario de práctica.
- Plantilla de ficha técnica.

## Formatos permitidos

- `DOCX` para documentos editables.
- `XLSX` para cálculos, costos, inventarios y fórmulas.
- `PPTX` para presentaciones o guías visuales editables.
- `PDF` para referencia no editable.
- `MD` para fuente editorial.

## Estructura mínima

1. Nombre de plantilla.
2. Propósito.
3. Capítulo o uso global.
4. Campos editables claramente identificados.
5. Instrucciones de llenado.
6. Ejemplo breve cuando sea necesario.
7. Versión y fecha.

## Reglas de actualización

- Mantener compatibilidad con la versión anterior cuando sea posible.
- Registrar cambios de campos obligatorios.
- No eliminar campos sin justificarlo en el historial del recurso.
- Publicar PDF de referencia cuando la plantilla editable pueda alterarse accidentalmente.
