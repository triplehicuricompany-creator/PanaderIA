# Evaluaciones Descargables

## Definición

Las evaluaciones son instrumentos para medir comprensión, desempeño práctico o avance del alumno en relación con los contenidos de PanaderIA™.

## Tipos de evaluación

- Autoevaluación.
- Examen de conocimiento.
- Rúbrica de práctica.
- Lista de cotejo evaluativa.
- Evaluación diagnóstica.
- Evaluación final de módulo.

## Formatos permitidos

- `PDF` para aplicación final.
- `DOCX` para edición académica.
- `XLSX` para matrices de calificación.
- `CSV` para bancos de preguntas o importación futura.
- `MD` para fuente editorial.

## Reglas pedagógicas

- Cada evaluación debe declarar objetivo, nivel esperado y criterios.
- Las preguntas deben corresponder directamente al capítulo o módulo.
- Las rúbricas deben incluir niveles de desempeño claros.
- Las respuestas o claves deben guardarse como recurso separado cuando se requiera confidencialidad.

## Control de versiones

Todo cambio en criterios, ponderaciones o respuestas correctas incrementa al menos versión menor. Correcciones de redacción sin impacto evaluativo pueden registrarse como parche.
