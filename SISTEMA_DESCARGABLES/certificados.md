# Certificados Descargables

## Definición

Los certificados son documentos de reconocimiento, constancia o finalización asociados a rutas, módulos, retos o logros definidos por PanaderIA™.

## Tipos permitidos

- Constancia de participación.
- Certificado de finalización de módulo.
- Certificado de ruta completa.
- Reconocimiento de práctica destacada.
- Diploma interno de logro.

## Formatos permitidos

- `PDF` como formato final oficial.
- `DOCX` o `PPTX` como fuente editable autorizada.
- `PNG` para previsualización o miniatura.
- `SVG` para elementos gráficos vectoriales autorizados.

## Campos mínimos

1. Nombre del alumno o participante.
2. Nombre del logro, módulo o ruta.
3. Fecha de emisión.
4. Código del certificado.
5. Versión del diseño.
6. Firma o sello autorizado cuando aplique.
7. Texto de validez o alcance.

## Reglas de control

- No se debe publicar certificado sin diseño aprobado.
- No se debe prometer acreditación externa si no existe convenio formal.
- Cada certificado debe tener código único o mecanismo de trazabilidad.
- Los certificados editables deben resguardarse para evitar uso no autorizado.
