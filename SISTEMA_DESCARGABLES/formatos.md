# Formatos Descargables

## Definición

Los formatos son documentos estructurados para capturar datos, evidencias, resultados o registros de producción y aprendizaje.

## Usos permitidos

- Registro de producción.
- Control de temperatura, tiempo y humedad.
- Registro de pruebas de masa.
- Control de inventario básico.
- Reportes de práctica.
- Bitácoras de mejora.

## Formatos permitidos

- `PDF` para impresión y archivo.
- `XLSX` para captura tabular o análisis.
- `CSV` para datos exportables.
- `DOCX` para reportes editables.
- `MD` para versión fuente.

## Estructura mínima

| Campo | Descripción |
|---|---|
| ID del formato | Código oficial según nomenclatura. |
| Capítulo | Capítulo o módulo relacionado. |
| Responsable | Persona que llena o revisa el formato. |
| Fecha | Fecha de uso o captura. |
| Datos principales | Campos medibles o descriptivos. |
| Observaciones | Hallazgos, ajustes o aprendizajes. |
| Validación | Firma, iniciales o aprobación cuando aplique. |

## Reglas de calidad

- Los campos deben ser claros, medibles y no redundantes.
- Todo formato debe indicar unidades de medida cuando aplique.
- Los formatos de cálculo deben proteger fórmulas en la versión final cuando sea posible.
- La versión PDF debe coincidir con la versión editable aprobada.
