# Hojas de Trabajo Descargables

## Definición

Las hojas de trabajo son recursos pedagógicos para practicar, reflexionar, resolver ejercicios o consolidar aprendizajes del Libro Maestro Premium PanaderIA™.

## Tipos

- Ejercicios de comprensión.
- Actividades de observación.
- Problemas de formulación.
- Casos prácticos.
- Mapas de proceso.
- Reflexiones guiadas.

## Formatos permitidos

- `PDF` para impresión y entrega al alumno.
- `DOCX` para edición por instructores.
- `XLSX` para ejercicios numéricos.
- `MD` para fuente editorial.

## Estructura mínima

1. Título.
2. Capítulo relacionado.
3. Objetivo de aprendizaje.
4. Instrucciones.
5. Actividad principal.
6. Espacio de respuesta.
7. Criterios de revisión o autoevaluación.
8. Versión.

## Relación con capítulos

Cada hoja de trabajo debe reforzar una sección concreta del capítulo. Si la actividad integra varios capítulos, se debe marcar como `CAP-GLOBAL` e indicar los capítulos secundarios en la ficha del recurso.
