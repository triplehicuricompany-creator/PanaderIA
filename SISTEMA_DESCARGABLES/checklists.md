# Checklists Descargables

## Definición

Los checklists son listas de verificación prácticas para confirmar que un proceso, preparación, diagnóstico o entrega cumple los criterios mínimos definidos por PanaderIA™.

## Usos permitidos

- Preparación antes de iniciar una receta o técnica.
- Verificación de mise en place.
- Control de fermentación, horneado, enfriado y empaque.
- Diagnóstico de errores comunes.
- Revisión previa a evaluación o práctica final.

## Formatos permitidos

- `PDF` para uso final imprimible.
- `DOCX` para edición controlada.
- `XLSX` cuando el checklist requiera cálculos, filtros o seguimiento.
- `MD` para versión fuente editorial.

## Estructura mínima

1. Título del checklist.
2. Código oficial del recurso.
3. Capítulo relacionado.
4. Objetivo de uso.
5. Instrucciones breves.
6. Lista de verificación con casillas.
7. Espacio para observaciones.
8. Versión y fecha.

## Reglas de calidad

- Cada punto debe ser observable y accionable.
- No deben mezclarse temas de capítulos distintos salvo que el recurso sea `CAP-GLOBAL`.
- Debe poder usarse sin explicación extensa adicional.
- Debe incluir criterios claros de aprobación o avance cuando aplique.
