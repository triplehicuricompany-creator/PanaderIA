# Control de Calidad de Descargables

## Objetivo

Garantizar que cada descargable de PanaderIA™ sea útil, claro, correcto, consistente y trazable antes de integrarse al Libro Maestro Premium PanaderIA™ o a una plataforma de distribución.

## Checklist maestro

### Relación editorial

- [ ] El recurso está vinculado a un capítulo, módulo o alcance global.
- [ ] El objetivo del recurso está definido.
- [ ] El recurso no modifica capítulos existentes.
- [ ] El recurso no contradice contenidos oficiales.

### Calidad pedagógica

- [ ] El recurso refuerza una habilidad, concepto o proceso concreto.
- [ ] Las instrucciones son comprensibles para el alumno objetivo.
- [ ] El nivel de dificultad corresponde al capítulo.
- [ ] Incluye ejemplos o criterios cuando son necesarios.

### Calidad técnica

- [ ] Las unidades, cantidades y términos técnicos son consistentes.
- [ ] Las tablas, fórmulas o cálculos fueron revisados.
- [ ] Los formatos editables funcionan correctamente.
- [ ] El PDF final conserva diseño, legibilidad y paginación.

### Nomenclatura y trazabilidad

- [ ] El ID cumple la convención oficial.
- [ ] El nombre de archivo no contiene espacios, acentos ni caracteres especiales.
- [ ] La versión está declarada.
- [ ] El estado editorial está registrado.
- [ ] La fecha de actualización está registrada.

### Publicación

- [ ] El formato final corresponde al uso previsto.
- [ ] Se generaron versiones fuente y exportadas cuando aplica.
- [ ] El recurso fue aprobado antes de publicarse.
- [ ] El índice o manifiesto de descargables fue actualizado.

## Criterio de aprobación

Un descargable solo puede cambiar a estado `aprobado` si cumple todos los puntos críticos de relación editorial, calidad técnica, nomenclatura y publicación. Cualquier punto no aplicable debe justificarse en la ficha del recurso.
