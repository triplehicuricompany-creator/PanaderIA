# Referencias Internas

## Propósito

Este documento establece cómo registrar referencias internas del Libro Maestro Premium PanaderIA™ entre capítulos, anexos, recursos multimedia, glosarios y rutas de aprendizaje.

## Tipos de referencias

- Capítulo a capítulo.
- Capítulo a anexo.
- Capítulo a recurso multimedia.
- Capítulo a glosario.
- Capítulo a checklist o descargable.
- Nota editorial a entrada estructural.
- Traducción a fuente original.

## Registro mínimo

Cada referencia debe incluir:

- Código de origen.
- Código de destino.
- Tipo de referencia.
- Propósito de la referencia.
- Estado de validación.
- Idioma o edición afectada.
- Observaciones editoriales.

## Reglas de trazabilidad

1. No crear referencias a contenido inexistente sin marcarlo como pendiente.
2. No usar números de página definitivos antes de la edición final.
3. Registrar referencias bidireccionales cuando el retorno sea necesario.
4. Verificar recursos externos antes de publicación futura.
5. Mantener separadas las referencias estructurales y las referencias bibliográficas.

## Relación con otros sistemas

Las referencias a imágenes, videos, audios y QR deben coordinarse con el Sistema Maestro Multimedia PanaderIA™. Las referencias necesarias para exportación deben coordinarse con el Sistema Maestro de Publicación PanaderIA™.
