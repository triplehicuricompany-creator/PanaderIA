# Sistema de Navegación

## Propósito

Este documento define las reglas de navegación interna del Libro Maestro Premium PanaderIA™ para lectura impresa, digital, web y multilenguaje.

## Tipos de navegación

- **Lineal:** lectura de principio a fin.
- **Modular:** acceso por capítulo, tema o bloque.
- **Técnica:** consulta por proceso, fórmula, problema o diagnóstico.
- **Multimedia:** acceso mediante QR, enlaces o recursos embebidos.
- **Multilenguaje:** cambio entre idiomas y variantes regionales.

## Elementos navegables

- Índice general.
- Índice por bloques o partes.
- Enlaces a capítulos relacionados.
- Referencias a anexos.
- Recursos descargables.
- Códigos QR.
- Glosario y tablas de consulta.
- Rutas de aprendizaje sugeridas.

## Reglas de navegación

1. Todo enlace interno debe apuntar a una entrada estructural existente.
2. Los enlaces no deben depender de páginas finales hasta que exista edición final.
3. Las rutas de lectura deben ser compatibles con impresión y digital.
4. Los QR y recursos externos deben registrarse en sistemas especializados.
5. La navegación debe evitar duplicar contenido del libro.

## Estados de enlaces

- `pendiente`
- `validado`
- `requiere_destino`
- `roto`
- `externo_en_revision`
- `archivado`
