# Preparación Multilenguaje

## Propósito

Este documento prepara el Libro Maestro Premium PanaderIA™ para futuras traducciones, localizaciones y ediciones regionales sin generar traducciones en esta misión.

## Idioma base

El idioma base editorial será el español, salvo que una edición futura autorizada defina otra fuente maestra.

## Componentes traducibles

- Títulos de capítulos y secciones.
- Texto editorial del libro.
- Tablas, glosarios y notas técnicas.
- Metadatos del libro.
- Textos alternativos de imágenes.
- Subtítulos o transcripciones de recursos multimedia.
- Materiales descargables.
- Navegación digital y etiquetas de interfaz.

## Campos por idioma

Cada entrada estructural deberá poder registrar:

- Idioma.
- Región o variante.
- Estado de traducción.
- Responsable o fuente.
- Fecha de revisión.
- Relación con la fuente maestra.
- Observaciones de localización.

## Estados de traducción

- `no_iniciado`
- `preparado_para_traduccion`
- `en_traduccion`
- `en_revision_tecnica`
- `en_revision_editorial`
- `aprobado`
- `requiere_actualizacion`
- `archivado`

## Reglas multilenguaje

1. No traducir contenido desde este sistema en esta misión.
2. Mantener identificadores estructurales independientes del idioma.
3. Registrar variantes regionales cuando afecten términos, medidas o ingredientes.
4. Coordinar unidades de medida, tecnicismos y nombres comerciales antes de publicar.
5. Mantener trazabilidad entre versión fuente y versión traducida.
