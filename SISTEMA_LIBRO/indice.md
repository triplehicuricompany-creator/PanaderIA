# Índice Maestro

## Propósito

El índice maestro es el mapa estructural del Libro Maestro Premium PanaderIA™. Su función es ordenar partes, capítulos, anexos y recursos sin escribir contenido nuevo.

## Componentes del índice

- Portada y preliminares.
- Introducción editorial.
- Partes o bloques principales.
- Capítulos.
- Secciones internas.
- Anexos.
- Glosario o recursos de consulta.
- Recursos multimedia vinculados.
- Cierre editorial.

## Campos de control

Cada entrada del índice debe poder registrar:

- Código estructural.
- Título visible.
- Nivel jerárquico.
- Archivo fuente relacionado, si existe.
- Estado editorial.
- Página o ubicación futura.
- Referencias cruzadas.
- Dependencia multilenguaje.

## Reglas del índice

1. El índice no valida contenido técnico por sí mismo.
2. Ninguna numeración final debe asumirse hasta revisión editorial.
3. Las entradas deben poder exportarse a formatos impresos y digitales.
4. Las referencias a páginas quedan como futuras hasta generar edición final.
5. El índice debe mantenerse sincronizado con capítulos, navegación y ediciones.

## Salidas futuras permitidas

En fases posteriores, el índice podrá alimentar tablas de contenido, navegación web, metadatos EPUB y controles de impresión. Esta misión no genera esas salidas.
