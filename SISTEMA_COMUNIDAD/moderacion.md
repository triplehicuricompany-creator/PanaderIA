# Moderación Conceptual

## Flujo futuro propuesto

1. Detectar contenido o conducta que requiere revisión.
2. Clasificar severidad de forma humana.
3. Escalar a soporte, legal, seguridad o gobernanza si aplica.
4. Registrar decisión documentalmente cuando exista sistema operativo autorizado.
5. Comunicar resolución con tono respetuoso y educativo.

## Límites

No se crean sanciones activas, bots de moderación, respuestas automáticas ni registros reales durante esta misión.
