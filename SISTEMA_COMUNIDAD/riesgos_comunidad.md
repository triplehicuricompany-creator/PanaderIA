# Riesgos de Comunidad

| Riesgo | Descripción | Mitigación conceptual |
| --- | --- | --- |
| Desinformación técnica | Recomendaciones incorrectas afectan aprendizaje. | Escalar a revisión educativa. |
| Conflictos entre usuarios | Conversaciones pierden tono respetuoso. | Moderación humana futura. |
| Exposición de datos | Alumnos comparten información sensible. | Políticas de privacidad claras. |
| Spam comercial | Publicaciones ajenas al propósito educativo. | Reglas y filtros futuros. |
| Sobrecarga de soporte | La comunidad intenta resolver incidencias privadas. | Separar canales de soporte. |

Estos riesgos no activan controles reales. Sirven para planear misiones futuras.
