# Roles Comunitarios Futuros

| Rol candidato | Función conceptual | Restricción |
| --- | --- | --- |
| Participante | Aprender, preguntar y compartir avances. | No registrado en esta misión. |
| Moderación | Cuidar convivencia y escalar incidencias. | No asignada. |
| Instructor | Aclarar dudas educativas autorizadas. | No operativo. |
| Soporte | Canalizar problemas técnicos o de acceso. | Depende de sistema de soporte. |
| Gobernanza | Resolver excepciones y cambios mayores. | Solo documental. |

Los roles son categorías de diseño, no nombramientos reales.
