# 07_PORTADAS_BANNERS

## Uso de esta carpeta

Portadas, banners, hero images y piezas promocionales premium.

## Reglas

- Usar nombres profesionales, descriptivos y consistentes.
- Revisar `../manifest.json` antes de agregar archivos para evitar duplicados.
- Registrar cada recurso importante en `../manifest.json`.
- No modificar el Libro Maestro, el Curso, Boly Assistant ni recursos externos desde esta carpeta.
- Si el recurso fue generado por IA y representa un proceso real del Chef Juanote, marcarlo como provisional y reemplazable por foto real.
- Mantener el estilo PanaderIA™: educativo, premium, amigable, mexicano, profesional y pensado para principiantes.

## Convención sugerida

```text
panaderia-07_portadas_banners-[tema]-[uso]-v01.[ext]
```

- [ ] ¿Evita duplicar un recurso existente?
- [ ] ¿Indica si es IA provisional o foto real?
