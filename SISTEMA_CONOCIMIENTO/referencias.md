# Referencias

Este documento define el marco para administrar referencias internas y externas del conocimiento PanaderIA™.

## Referencias internas

Conexiones hacia documentos, sistemas, mapas, manifiestos, bibliotecas o recursos propios del ecosistema.

## Referencias externas

Conexiones hacia materiales, normas, publicaciones o recursos ajenos al ecosistema que puedan respaldar o contextualizar conocimiento.

## Criterios de referencia

- Toda referencia debe tener un propósito claro.
- Toda referencia debe estar asociada a un dominio o subdominio.
- Las referencias críticas deben revisarse periódicamente.
- Las referencias externas no sustituyen la validación editorial o técnica interna.

## Estructura recomendada

- Código de referencia.
- Título.
- Tipo.
- Relación con el conocimiento PanaderIA™.
- Dominio vinculado.
- Estado.
- Fecha de revisión.
