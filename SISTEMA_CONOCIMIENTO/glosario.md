# Glosario oficial PanaderIA™

Este glosario establece términos base del ecosistema. Las definiciones son estructurales y no sustituyen contenidos técnicos, pedagógicos o editoriales desarrollados.

## Términos principales

### PanaderIA™
Ecosistema integral de conocimiento, producción, documentación, asistencia y desarrollo alrededor de la panadería.

### Sistema Maestro de Conocimiento
Infraestructura documental encargada de organizar el conocimiento oficial y sus relaciones.

### Conocimiento canónico
Información aprobada como referencia oficial para el ecosistema.

### Dominio
Categoría mayor de conocimiento dentro del mapa general.

### Subdominio
Categoría específica dependiente de un dominio.

### Fuente
Origen de información registrado para fines de soporte, contraste o validación.

### Referencia
Vínculo conceptual, documental o metodológico que conecta elementos de conocimiento.

### Terminología controlada
Lista de términos recomendados, permitidos o reservados para asegurar consistencia.

### Gobernanza del conocimiento
Reglas para crear, actualizar, aprobar, relacionar y retirar elementos de conocimiento.

### Roadmap de conocimiento
Plan progresivo de crecimiento, revisión y maduración del sistema.
