# Fuentes

Este archivo define la estructura oficial para registrar fuentes dentro del Sistema Maestro de Conocimiento PanaderIA™.

## Propósito

Permitir que cualquier elemento de conocimiento pueda vincularse con un origen identificable, evaluable y trazable.

## Tipos de fuentes

- Fuentes internas del ecosistema PanaderIA™.
- Documentos maestros aprobados.
- Sistemas documentales internos.
- Referencias técnicas externas.
- Referencias normativas externas.
- Bibliografía especializada.
- Evidencia operativa documentada.

## Campos recomendados para registrar una fuente

- Identificador de fuente.
- Nombre de la fuente.
- Tipo de fuente.
- Autoría o responsable.
- Fecha de registro.
- Estado de validación.
- Dominio relacionado.
- Observaciones.

## Estados de validación

- `pendiente`
- `validada`
- `en_revision`
- `rechazada`
- `archivada`
