# Sistema Maestro de Conocimiento PanaderIA™

Este directorio contiene la infraestructura documental oficial para administrar, ordenar y evolucionar el conocimiento del ecosistema PanaderIA™.

## Propósito

El Sistema Maestro de Conocimiento PanaderIA™ define el marco de gobernanza, clasificación, consulta y crecimiento del conocimiento institucional, técnico, editorial, pedagógico y operativo del proyecto.

## Alcance

Este sistema no contiene capítulos del libro, cursos, respuestas técnicas desarrolladas ni materiales finales para alumnos. Su función es organizar la infraestructura documental que permitirá mantener consistencia, trazabilidad y alineación entre los distintos sistemas de PanaderIA™.

## Componentes

- `manifest.json`: Identidad, alcance y estructura del sistema.
- `conceptos.md`: Conceptos fundamentales del sistema de conocimiento.
- `glosario.md`: Glosario oficial del ecosistema.
- `terminologia.md`: Terminología técnica y editorial controlada.
- `fuentes.md`: Registro estructural de fuentes admitidas.
- `referencias.md`: Marco de referencias internas y externas.
- `faq.md`: Estructura base de preguntas frecuentes.
- `mapa_conocimiento.md`: Mapa general de dominios de conocimiento.
- `roadmap.md`: Roadmap de crecimiento del sistema.

## Principios rectores

1. Centralización del conocimiento canónico.
2. Separación entre infraestructura documental y contenido final.
3. Trazabilidad de conceptos, fuentes y referencias.
4. Terminología consistente en todo el ecosistema.
5. Escalabilidad para futuras fases editoriales, técnicas y pedagógicas.
