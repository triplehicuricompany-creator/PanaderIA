# Terminología técnica y editorial

Este documento regula la terminología base que debe emplearse en el Sistema Maestro de Conocimiento PanaderIA™.

## Criterios de uso

- Usar términos consistentes entre sistemas documentales.
- Evitar duplicidades semánticas sin justificación.
- Separar términos técnicos, editoriales, pedagógicos, comerciales y operativos.
- Registrar términos nuevos antes de incorporarlos como canónicos.

## Categorías terminológicas

### Terminología institucional
Términos vinculados con identidad, sistemas, gobernanza y arquitectura del ecosistema.

### Terminología técnica
Términos asociados a procesos, métodos, herramientas o variables especializadas.

### Terminología editorial
Términos usados para organizar documentos, capítulos, versiones, recursos y publicaciones.

### Terminología pedagógica
Términos relacionados con aprendizaje, evaluación, acompañamiento y estructura didáctica.

### Terminología operativa
Términos relacionados con producción, control, administración y procesos internos.

## Estados de un término

- `propuesto`: término en evaluación.
- `aprobado`: término autorizado para uso oficial.
- `reservado`: término protegido para un uso específico.
- `obsoleto`: término retirado o reemplazado.
- `pendiente_revision`: término que requiere validación.
