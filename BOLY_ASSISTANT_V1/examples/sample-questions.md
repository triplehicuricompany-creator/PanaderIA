# Preguntas de ejemplo para Boly™ V1

```bash
npm run ask -- "Boly, mi masa quedó pegajosa durante el amasado"
npm run ask -- "¿Por qué mi bolillo no abre en el corte?"
npm run ask -- "Mi corteza queda pálida y blanda"
npm run ask -- "Quiero hacer bolillo integral, ¿qué ajusto?"
npm run ask -- "¿Cómo sé si mi fermentación ya está lista?"
npm run ask -- "¿Qué necesito para certificarme?"
npm run ask -- "¿Cómo calculo una receta de bolillo?"
```

## Preguntas útiles para recopilar más contexto

- ¿Qué fórmula usaste?
- ¿Qué harina compraste?
- ¿Cuánta agua agregaste?
- ¿Cuánto tiempo amasaste?
- ¿La masa pasó prueba de ventana?
- ¿Cuánto tiempo fermentó?
- ¿A qué temperatura estaba tu cocina?
- ¿Usaste vapor en el horno?
- ¿Enfriaste el pan sobre rejilla?
- ¿Tienes foto del corte, miga y corteza?
